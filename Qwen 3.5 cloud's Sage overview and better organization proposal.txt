### 2. Plan to Simplify & Streamline (Refactoring)

Do not rewrite everything at once. Use the **Strangler Fig Pattern**: gradually replace pieces of the monolith with modular components.

#### Step 1: Enforce Modularization
Break `Sage Lite.py` into a package structure.
```text
Sage/
├── main.py              # Entry point only
├── sage_config.json
├── core/
│   ├── __init__.py
│   ├── config.py        # Config loading & validation (Pydantic)
│   ├── state.py         # Global state management (replace globals)
│   └── router.py        # Intent parsing & command routing
├── agents/
│   ├── orchestrator.py  # Model selection logic
│   ├── proactive.py     # Background sentinels
│   └── memory.py        # History & persistent context
├── tools/
│   ├── files.py         # OS operations (mkdir, delete, search)
│   ├── reminders.py     # Reminder logic
│   └── web.py           # Serper, Weather, Transit
├── voice/
│   ├── stt.py           # Speech-to-Text (Whisper sidecar)
│   ├── tts.py           # Text-to-Speech (Kokoro)
│   └── wake.py          # Wake word detection
└── ui/
    ├── tk_window.py     # Tkinter logic
    └── widgets.py       # Custom UI components
```

#### Step 2: Standardize Logging
Replace `print` and manual file writing with Python's `logging` module.
*   **Why:** You can set levels (DEBUG, INFO, ERROR) dynamically. You can log to file and console simultaneously.
*   **Action:** Create a `logger` in `core/state.py` and import it everywhere.

#### Step 3: Command Registry Pattern
Instead of the massive `COMMANDS` dictionary and `quick_route` regex spaghetti, use a decorator registry.
```python
# core/router.py
command_registry = {}

def register_command(name, aliases=None):
    def decorator(func):
        command_registry[name] = func
        if aliases:
            for alias in aliases:
                command_registry[alias] = func
        return func
    return decorator

@register_command("remind", aliases=["reminder", "alarm"])
def handle_remind(args):
    ...
```
This makes adding new commands trivial and keeps routing logic separate from execution logic.

#### Step 4: Validate Configuration
Use **Pydantic** to validate `sage_config.json` on load.
*   **Why:** Currently, if you typo a key in JSON, the script might crash 20 minutes later when that key is accessed. Pydantic will fail immediately on startup with a clear error message.
*   **Benefit:** You get type safety (e.g., ensuring `timeout_seconds` is actually an int).

#### Step 5: Decouple UI from Logic
The Tkinter code should not know *how* a reminder is saved, only *that* it was saved.
*   **Action:** The UI should call `agents.proactive.add_reminder()`. It should not import `handle_remind_add_precise` directly. This allows you to swap Tkinter for a Web UI or CLI later without breaking the backend.

### 3. Ideas to Develop Further (Evolution)

#### A. True Function Calling (Tool Use)
Currently, you use regex and a secondary LLM call (`parse_intent`) to figure out if the user wants a reminder.
*   **Upgrade:** Modern Ollama models (Llama 3.1, Qwen 2.5) support **JSON Mode** or **Tool Calling**.
*   **Implementation:** Define your tools (remind, note, search) in the system prompt as JSON schemas. Ask the model to output *only* JSON.
*   **Benefit:** Eliminates the fragile `quick_route` regex parsing and reduces latency by removing the "intent parser" model call.

#### B. Vector Database for Memory
You are currently using keyword matching (`retrieve_relevant_memories`) and sliding windows.
*   **Upgrade:** Integrate a lightweight local vector DB like **LanceDB** or **ChromaDB**.
*   **Implementation:** When `mem-add` is called, embed the text and store it. When chatting, query the DB for semantic similarity.
*   **Benefit:** You can ask "What did I say about the insurance last month?" and it will find it even if you don't use the exact keyword "insurance".

#### C. Local API Server (FastAPI)
Currently, Sage is a desktop app.
*   **Upgrade:** Run a hidden `FastAPI` server in the background (`main.py`).
*   **Benefit:**
    *   You can trigger Sage from other apps (e.g., a browser extension, a hotkey script).
    *   You can separate the UI process from the Logic process. If the UI crashes, the background agents (reminders, wake word) keep running.

#### D. Dockerized Sidecars
Your voice and OCR sidecars rely on specific Python environments (`.venv`).
*   **Upgrade:** Containerize the Whisper and OCR services using Docker.
*   **Benefit:** Isolates heavy dependencies (CUDA, ffmpeg, ONNX) from your main Sage environment. Makes deployment on other machines easier.

### 4. Specific Code Critiques & Fixes

#### 1. Global Config Access
**Current:** `CONFIG = load_config()` at top level, accessed globally.
**Fix:** Pass config as an argument or use a Singleton class.
```python
# core/state.py
class AppState:
    _instance = None
    def __init__(self):
        self.config = load_config()
        self.memory = []

    @classmethod
    def get(cls):
        if not cls._instance:
            cls._instance = AppState()
        return cls._instance
```

#### 2. Thread Safety
**Current:** You use `threading.Lock()` in many places, but global variables like `current_mode` are accessed without locks in some UI callbacks.
**Fix:** Ensure *all* shared state access is protected by locks or use a thread-safe queue for UI updates.
```python
# In UI class
def log(self, message):
    # Schedule UI update on main thread to prevent Tkinter threading errors
    self.root.after(0, lambda: self._safe_log(message))
```

#### 3. Hardcoded Paths
**Current:** `WORKSPACE_ROOT = Path(__file__).resolve().parent` and specific Windows paths (`D:\\SageBackup`).
**Fix:** Use `pathlib` exclusively and resolve paths relative to a configurable `BASE_DIR` in JSON. Avoid `D:\\` hardcoded strings; use environment variables or config.

#### 4. Error Handling
**Current:** Many `try...except Exception: pass` blocks.
**Fix:** Never silence exceptions without logging them.
```python
# Bad
try:
    do_something()
except Exception:
    pass

# Good
try:
    do_something()
except Exception as e:
    logger.error(f"Failed to do_something: {e}", exc_info=True)
```

```python
class ConfigManager:
    """Manages configuration loading, validation, and saving."""

    DEFAULT_CONFIG_PATH = "sage_config.json"

    def __init__(self, config_path: Optional[str] = None):
        self.config_path = Path(config_path or self.DEFAULT_CONFIG_PATH)
        self._config: Optional[SageConfig] = None
        self._last_loaded: Optional[datetime] = None

    def load(self, force: bool = False) -> SageConfig:
        """Load and validate configuration from JSON file."""
        if not force and self._config is not None:
            return self._config

        if not self.config_path.exists():
            raise FileNotFoundError(f"Configuration file not found: {self.config_path}")

        try:
            with open(self.config_path, 'r', encoding='utf-8') as f:
                raw_data = json.load(f)

            self._config = SageConfig(**raw_data)
            self._last_loaded = datetime.now()
            return self._config

        except json.JSONDecodeError as e:
            raise ValueError(f"Invalid JSON in config file: {e}")
        except ValidationError as e:
            raise ValueError(f"Configuration validation failed: {e}")

    def save(self, config: Optional[SageConfig] = None) -> bool:
        """Save configuration to JSON file."""
        try:
            cfg = config or self._config
            if cfg is None:
                return False

            with open(self.config_path, 'w', encoding='utf-8') as f:
                json.dump(cfg.dict(), f, indent=2, ensure_ascii=False)
                f.write("\n")

            self._config = cfg
            self._last_loaded = datetime.now()
            return True

        except Exception as e:
            print(f"[ERROR] Failed to save config: {e}")
            return False

    def get(self, section: str, key: Optional[str] = None, default: Any = None) -> Any:
        """Get a configuration value by section and key."""
        if self._config is None:
            self.load()

        try:
            section_data = getattr(self._config, section)
            if key is None:
                return section_data
            if isinstance(section_data, dict):
                return section_data.get(key, default)
            return getattr(section_data, key, default)
        except (AttributeError, KeyError):
            return default

    def update(self, section: str, key: str, value: Any, persist: bool = True) -> bool:
        """Update a configuration value."""
        if self._config is None:
            self.load()

        try:
            section_data = getattr(self._config, section)
            if isinstance(section_data, dict):
                section_data[key] = value
            else:
                setattr(section_data, key, value)

            if persist:
                return self.save()
            return True

        except Exception as e:
            print(f"[ERROR] Failed to update config: {e}")
            return False

    @property
    def config(self) -> SageConfig:
        """Get the current configuration."""
        if self._config is None:
            self.load()
        return self._config

    @property
    def last_loaded(self) -> Optional[datetime]:
        """Get the timestamp of the last config load."""
        return self._last_loaded
```

```python
class AppState:
    """
    Thread-safe singleton for managing application state.
    Replaces all global variables with proper state management.
    """

    _instance: Optional['AppState'] = None
    _lock = threading.Lock()

    def __new__(cls) -> 'AppState':
        if cls._instance is None:
            with cls._lock:
                if cls._instance is None:
                    cls._instance = super().__new__(cls)
                    cls._instance._initialized = False
        return cls._instance

    def __init__(self):
        if self._initialized:
            return

        self._state_lock = threading.RLock()

        self._mode: AppMode = AppMode.ADMIN

        """Check if a reminder is a recent duplicate."""

            now_ts = time.time()

            stale_keys = [k for k, ts in self._reminder_recent_adds.items() if ts < cutoff]
            for stale in stale_keys:
                del self._reminder_recent_adds[stale]

            # Check for duplicate
            last = self._reminder_recent_adds.get(key)
            if last is not None and (now_ts - last) <= window_seconds:
                return True

            # Record new reminder
            self._reminder_recent_adds[key] = now_ts
            return False
```

```python
# -*- coding: utf-8 -*-
"""
Sage Core Utilities
Common helper functions extracted from the monolith.
"""

import re
from typing import Optional


def _history_safe_text(value: str, max_chars: int = 1400) -> str:
    """Truncate text for history storage."""
    text = re.sub(r"\s+", " ", str(value or "")).strip()
    if len(text) <= max_chars:
        return text
    return text[:max_chars].rstrip() + " …[truncated]"


def _context_summary(text: str, max_chars: int = 180) -> str:
    """Create a summary of model output for context stack."""
    value = re.sub(r"\s+", " ", str(text or "")).strip()
    if not value:
        return "(no response)"

    # Keep short-term recall crisp: first sentence or truncated clause.
    m = re.search(r"[.!?]", value)
    if m and m.start() >= 24:
        value = value[: m.start() + 1]

    if len(value) > max_chars:
        value = value[:max_chars].rstrip() + "..."

    return value


def _normalize_text_for_tts(text: str) -> str:
    """Normalize markdown-ish text into clean spoken text for TTS engines."""
    value = str(text or "").strip()
    if not value:
        return ""

    # Convert common LaTeX/KaTeX fragments into plain spoken-friendly text.
    value = re.sub(r"\$\$(.+?)\$\$", r" \1 ", value, flags=re.DOTALL)
    value = re.sub(r"\$(.+?)\$", r" \1 ", value, flags=re.DOTALL)

    # Convert markdown structure to natural pauses.
    value = re.sub(r"(?m)^\s*#{1,6}\s+", "", value)
    value = re.sub(r"(?m)^\s*[-*]\s+", "- ", value)
    value = re.sub(r"(?m)^\s*\d+\.\s+", "", value)

    # Remove emphasis/backticks
    value = value.replace("**", "").replace("__", "").replace("`", "").replace("*", "")

    # Expand arithmetic operators for speech
    value = re.sub(r"(?<=\d)\s*\+\s*(?=\d)", " plus ", value)
    value = re.sub(r"(?<=\d)\s*\-\s*(?=\d)", " minus ", value)
    value = re.sub(r"(?<=\d)\s*=\s*(?=\d)", " equals ", value)
    value = re.sub(r"(?<=\d)\s*/\s*(?=\d)", " divided by ", value)

    value = re.sub(r"\s+", " ", value).strip()
    return value


def _validate_hhmm(text: str) -> Optional[str]:
    """Validate and normalize HH:MM time format."""
    raw = str(text or "").strip()
    m = re.match(r"^(\d{1,2}):(\d{2})$", raw)
    if not m:
        return None
    hh = int(m.group(1))
    mm = int(m.group(2))
    if hh < 0 or hh > 23 or mm < 0 or mm > 59:
        return None
    return f"{hh:02d}:{mm:02d}"


def _strip_internal_warning_artifacts(text: str) -> str:
    """Remove known internal warning tokens accidentally echoed inside normal answers."""
    value = str(text or "")
    if not value:
        return ""

    patterns = [
        r"\[WARNING\]\s*Deep model returned an empty response\.?(?:\s*)",
        r"\[WARNING\]\s*Deep raw call timed out after retry\.?(?:\s*)",
        r"\[WARNING\]\s*Deep raw stream interrupted before completion\.?(?:\s*)",
    ]
    cleaned = value
    for pat in patterns:
        cleaned = re.sub(pat, " ", cleaned, flags=re.IGNORECASE)

    cleaned = re.sub(r"\s{2,}", " ", cleaned)
    cleaned = re.sub(r"\n[ \t]+", "\n", cleaned)
    cleaned = re.sub(r"[ \t]+\n", "\n", cleaned)
    return cleaned.strip()
```

```python
# -*- coding: utf-8 -*-
"""
Sage Lite - Main Entry Point
"""

import sys
import logging
from pathlib import Path

# Initialize logging first
logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s | %(levelname)s | %(name)s | %(message)s',
    handlers=[
        logging.FileHandler('NOTES/sage.log', encoding='utf-8'),
        logging.StreamHandler()
    ]
)
logger = logging.getLogger(__name__)


def main():
    """Main entry point for Sage."""
    logger.info("Sage Lite starting...")

    # Initialize core state
    from core.state import state
    from core.config import config_manager

    try:
        # Load configuration
        config = config_manager.load()
        logger.info(f"Configuration loaded from {config_manager.config_path}")

        # Check for CLI arguments
        if len(sys.argv) > 1:
            if sys.argv[1] == "--ui":
                from ui.tk_window import launch_ui
                return launch_ui()
            elif sys.argv[1] == "--monitor":
                from tools.reminders import monitor_reminders_precise
                return monitor_reminders_precise()

        # Default: CLI mode
        from cli import run_cli
        return run_cli()

    except Exception as e:
        logger.error(f"Failed to start Sage: {e}", exc_info=True)
        return 1
    finally:
        # Cleanup on exit
        from core.shutdown import shutdown_background_services
        shutdown_background_services()


if __name__ == "__main__":
    sys.exit(main())
```

```python
# -*- coding: utf-8 -*-
"""
Sage Command Router
Handles intent parsing, command registration, and execution routing.
Replaces the monolithic quick_route and COMMANDS dictionary.
"""

import re
import json
import logging
import shlex
from datetime import datetime, timedelta
from typing import Any, Callable, Dict, List, Optional, Tuple
from dataclasses import dataclass
from enum import Enum
from pathlib import Path

from .state import state, AppState
from .config import get_config
from .utils import _history_safe_text

logger = logging.getLogger(__name__)


class CommandType(Enum):
    """Types of commands in the routing system."""
    CORE = "core"
    SKILL = "skill"
    VOICE = "voice"
    SYSTEM = "system"
    CHAT = "chat"


@dataclass
class CommandRegistration:
    """Registration metadata for a command."""
    name: str
    handler: Callable
    aliases: List[str]
    command_type: CommandType
    description: str
    modes: List[str]  # Which modes can execute this (admin, chat, both)
    requires_args: bool = False
    lite_enabled: bool = True


class CommandRegistry:
    """
    Central registry for all Sage commands.
    Uses decorator pattern for easy command registration.
    """

    _instance: Optional['CommandRegistry'] = None
    _lock = None

    def __new__(cls) -> 'CommandRegistry':
        if cls._instance is None:
            cls._instance = super().__new__(cls)
            cls._instance._commands: Dict[str, CommandRegistration] = {}
            cls._instance._lite_commands: set = set()
        return cls._instance

    def register(
        self,
        name: str,
        aliases: Optional[List[str]] = None,
        command_type: CommandType = CommandType.CORE,
        description: str = "",
        modes: Optional[List[str]] = None,
        requires_args: bool = False,
        lite_enabled: bool = True,
    ):
        """
        Decorator to register a command handler.

        Example:
            @registry.register("remind", aliases=["reminder", "alarm"], description="Add a reminder")
            def handle_remind(args):
                ...
        """
        def decorator(handler: Callable):
            reg = CommandRegistration(
                name=name,
                handler=handler,
                aliases=aliases or [],
                command_type=command_type,
                description=description,
                modes=modes or ["admin", "chat"],
                requires_args=requires_args,
                lite_enabled=lite_enabled,
            )

            # Register main name
            self._commands[name] = reg

            # Register aliases
            for alias in (aliases or []):
                self._commands[alias] = reg

            # Track lite-enabled commands
            if lite_enabled:
                self._lite_commands.add(name)

            logger.debug(f"Registered command: {name} (aliases: {aliases or []})")
            return handler

        return decorator

    def get(self, command_name: str) -> Optional[CommandRegistration]:
        """Get a command registration by name or alias."""
        return self._commands.get(command_name.lower())

    def list_commands(self, command_type: Optional[CommandType] = None, mode: Optional[str] = None) -> List[Dict[str, Any]]:
        """List all registered commands, optionally filtered."""
        commands = []
        for name, reg in self._commands.items():
            # Skip aliases (only show main command names)
            if reg.name != name:
                continue

            # Filter by type
            if command_type and reg.command_type != command_type:
                continue

            # Filter by mode
            if mode and mode not in reg.modes:
                continue

            commands.append({
                "name": reg.name,
                "aliases": reg.aliases,
                "type": reg.command_type.value,
                "description": reg.description,
                "requires_args": reg.requires_args,
            })

        # Sort by name
        commands.sort(key=lambda x: x["name"])
        return commands

    def is_lite_command(self, command_name: str) -> bool:
        """Check if a command is enabled in lite mode."""
        return command_name.lower() in self._lite_commands

def _normalize_workspace_name(name: str) -> Optional[str]:
    """Normalize a workspace name to a known profile."""
    if not isinstance(name, str):
        return None

    value = name.strip().lower()
    if not value:
        return None

    known_workspaces = {"main", "admin", "health", "home", "project", "beba", "bajka"}

    if value == "default":
        return "main"
    if value in known_workspaces:
        return value

    return None


def _extract_workspace_targets(raw_text: str, orchestrator_data: Optional[Dict] = None) -> List[str]:
    """Extract workspace targets from user text."""
    targets: List[str] = []
    lowered = (raw_text or "").lower()

    def add_target(ws_name: Optional[str]):
        if ws_name and ws_name not in targets:
            targets.append(ws_name)

    # Check orchestrator data first
    if isinstance(orchestrator_data, dict):
        ws_from_orch = _normalize_workspace_name(orchestrator_data.get("workspace"))
        if ws_from_orch:
            add_target(ws_from_orch)

        ws_many = orchestrator_data.get("workspaces")
        if isinstance(ws_many, list):
            for candidate in ws_many:
                add_target(_normalize_workspace_name(candidate))

    # Check for explicit workspace keywords
    has_ws_keyword = bool(re.search(r"\b(workspace|workspaces|ws)\b", lowered))

    known_workspaces = ["admin", "health", "home", "project", "beba", "bajka", "main"]

    if has_ws_keyword:
        for ws_name in known_workspaces:
            if re.search(rf"\b{re.escape(ws_name)}\b", lowered):
                add_target(ws_name)
    else:
        for ws_name in known_workspaces:
            if re.search(rf"\b(?:to|in|into|for|under)\s+{re.escape(ws_name)}\b", lowered):
                add_target(ws_name)

    # Fallback to keyword-based inference
    if not targets:
        workspace_keyword_map = {
            "health": ["clinic", "hospital", "doctor", "medical", "pharmacy", "appointment"],
            "admin": ["tax", "form", "bill", "receipt", "bank", "payment"],
            "home": ["home", "house", "kitchen", "groceries", "cleaning", "repair"],
            "project": ["project", "deadline", "task", "coding", "deploy", "bug"],
            "beba": ["beba", "baby", "infant", "pediatric", "diaper"],
            "bajka": ["bajka", "story", "book", "reading", "bedtime"],
        }

        workspace_scores = {}
        for ws_name, keywords in workspace_keyword_map.items():
            score = sum(1 for word in keywords if re.search(rf"\b{re.escape(word)}\b", lowered))
            if score > 0:
                workspace_scores[ws_name] = score

        if workspace_scores:
            best_workspace = max(workspace_scores, key=workspace_scores.get)
            add_target(best_workspace)

    # Default to current workspace
    if not targets:
        current = _normalize_workspace_name(state.current_workspace)
        targets = [current or "main"]

    return targets


def _strip_workspace_mentions(raw_text: str, workspaces: List[str]) -> str:
    """Remove workspace mentions from text after extraction."""
    cleaned = raw_text or ""
    for ws_name in workspaces:
        cleaned = re.sub(rf"\b(?:to|in|into|for|under)\s+{re.escape(ws_name)}\b", " ", cleaned, flags=re.IGNORECASE)
        cleaned = re.sub(rf"\b{re.escape(ws_name)}\s+(?:workspace|ws)\b", " ", cleaned, flags=re.IGNORECASE)
        cleaned = re.sub(rf"\b{re.escape(ws_name)}\b", " ", cleaned, flags=re.IGNORECASE)

    cleaned = re.sub(r"\bworkspace\b|\bworkspaces\b|\bws\b", " ", cleaned, flags=re.IGNORECASE)
    cleaned = re.sub(r"\s+", " ", cleaned).strip(" .:-")
    return cleaned


def run_with_workspace_routing(
    target_workspaces: List[str],
    action_func: Callable[[str], str],
) -> str:
    """
    Execute an action across multiple workspaces.

    Args:
        target_workspaces: List of workspace names to route to
        action_func: Function that takes workspace name and returns result string

    Returns:
        Combined output from all workspace executions
    """
    original_workspace = _normalize_workspace_name(state.current_workspace) or "main"

    # Deduplicate and normalize targets
    unique_targets = []
    for ws_name in target_workspaces:
        normalized = _normalize_workspace_name(ws_name)
        if normalized and normalized not in unique_targets:
            unique_targets.append(normalized)

    if not unique_targets:
        unique_targets = [original_workspace]

    logger.debug(f"Workspace routing: {unique_targets} (original: {original_workspace})")

    outputs = []
    for ws_name in unique_targets:
        try:
            state.current_workspace = ws_name
            result = action_func(ws_name)

            if len(unique_targets) == 1:
                outputs.append(result)
            else:
                outputs.append(f"[{ws_name}] {result}")

        except Exception as e:
            logger.error(f"Workspace routing failed for {ws_name}: {e}")
            outputs.append(f"[{ws_name}] ERROR: {e}")
        finally:
            # Restore original workspace
            if state.current_workspace != original_workspace:
                state.current_workspace = original_workspace

    return "\n".join(outputs)

def _normalize_time_or_default(value: Optional[str], fallback: Optional[str] = None) -> str:
    """Normalize time to HH:MM format or return default."""
    from .utils import _validate_hhmm

    if not fallback:
        # Get default from config
        config = get_config()
        fallback = config.reminders.fixed_bell_time if config.reminders.use_fixed_bell_time else "08:00"

    candidate = str(value or "").strip()
    if not candidate:
        return fallback

    validated = _validate_hhmm(candidate)
    return validated if validated else fallback


def _resolve_day_token(day_token: str) -> Optional[datetime]:
    """Resolve a day token to a datetime object."""
    token = str(day_token or "").strip().lower()
    base_date = datetime.now().date()

    if token == "today":
        return datetime.combine(base_date, datetime.min.time())

    if token == "tomorrow":
        return datetime.combine(base_date + timedelta(days=1), datetime.min.time())

    if token == "next week":
        return datetime.combine(base_date + timedelta(days=7), datetime.min.time())

    weekday_map = {
        "monday": 0, "tuesday": 1, "wednesday": 2,
        "thursday": 3, "friday": 4, "saturday": 5, "sunday": 6,
    }

    # Next weekday
    m = re.match(r"^next\s+(monday|tuesday|wednesday|thursday|friday|saturday|sunday)$", token)
    if m:
        target = weekday_map[m.group(1)]
        today_wd = base_date.weekday()
        days_ahead = (target - today_wd) % 7
        if days_ahead == 0:
            days_ahead = 7
        return datetime.combine(base_date + timedelta(days=days_ahead), datetime.min.time())

    # This weekday
    m = re.match(r"^this\s+(monday|tuesday|wednesday|thursday|friday|saturday|sunday)$", token)
    if m:
        target = weekday_map[m.group(1)]
        today_wd = base_date.weekday()
        days_ahead = (target - today_wd) % 7
        return datetime.combine(base_date + timedelta(days=days_ahead), datetime.min.time())

    # In N days
    m = re.match(r"^in\s+(\d{1,3})\s+day(?:s)?$", token)
    if m:
        try:
            offset = int(m.group(1))
            if offset >= 0:
                return datetime.combine(base_date + timedelta(days=offset), datetime.min.time())
        except Exception:
            pass

    return None


def _parse_reminder_command(raw_text: str) -> Optional[Dict[str, Any]]:
    """Parse a reminder command from natural language."""
    text = raw_text.lower().replace("remind me", "").replace("remind", "").strip()

    if text == "list":
        return {"cmd": "remind", "args": ["list"]}

    if text.startswith("dedupe"):
        extra = text.split()[1:]
        return {"cmd": "remind", "args": ["dedupe", *extra]}

    # Look for date pattern YYYY-MM-DD
    date_match = re.search(r'\d{4}-\d{2}-\d{2}', text)
    if date_match:
        date = date_match.group(0)
        desc = text[date_match.end():].strip(' -:')
        if desc:
            return {"cmd": "remind", "args": [date, desc]}

    # Fallback
    return {"cmd": "remind", "args": [text]} if text else {"cmd": "remind", "args": []}


def _detect_brainstorm_keywords(user_text: str) -> bool:
    """Detect if user is expressing wishes/needs for new features."""
    brainstorm_phrases = [
        'i wish', 'would be nice', 'i need', 'would be great',
        'could you', 'can you add', 'would love', 'wish i could',
        'if only', 'hope to', 'want to be able'
    ]
    text_lower = user_text.lower()
    return any(phrase in text_lower for phrase in brainstorm_phrases)


# ==================== ROUTER EXECUTION ====================

def route_command(user_input: str, mode: Optional[str] = None) -> Tuple[str, bool]:
    """
    Route a user input to the appropriate command handler.

    Args:
        user_input: The raw user input text
        mode: Optional mode override ("admin" or "chat")

    Returns:
        Tuple of (response_text, was_handled)
    """
    from .state import state

    # Use current mode if not specified
    if mode is None:
        mode = state.mode.value

    # Try quick route first
    parsed = quick_route(user_input)

    if parsed is None:
        # No quick match - needs LLM parsing
        return "", False

    cmd = parsed.get("cmd", "").lower()
    args = parsed.get("args", [])

    # Check if command exists in registry
    reg = registry.get(cmd)

    if reg is None:
        # Command not registered - return as chat
        return "", False

    # Check mode compatibility
    if mode not in reg.modes:
        logger.warning(f"Command {cmd} not allowed in {mode} mode")
        return f"[BLOCKED] This command is not available in {mode} mode.", True

    # Check lite profile
    config = get_config()
    is_lite = config.active_profile.lower() in {"lite", "mini"}

    if is_lite and not reg.lite_enabled:
        return "[LITE] This command is disabled in Lite profile. Switch to full profile to enable it.", True

    # Execute the command
    try:
        logger.debug(f"Executing command: {cmd} with args: {args}")
        result = reg.handler(args)
        return result, True
    except Exception as e:
        logger.error(f"Command execution failed: {cmd} - {e}", exc_info=True)
        return f"[ERROR] Command '{cmd}' failed: {e}", True


def get_command_help(command_name: str) -> Optional[str]:
    """Get help text for a specific command."""
    reg = registry.get(command_name)
    if reg is None:
        return None

    help_text = f"**{reg.name}**"
    if reg.aliases:
        help_text += f" (aliases: {', '.join(reg.aliases)})"
    help_text += f"\n\n{reg.description}"
    help_text += f"\n\nType: {reg.command_type.value}"
    help_text += f"\nModes: {', '.join(reg.modes)}"
    if reg.requires_args:
        help_text += "\n\nRequires arguments"

    return help_text


def list_available_commands(mode: Optional[str] = None, lite_only: bool = False) -> str:
    """Generate a formatted list of available commands."""
    config = get_config()
    is_lite = lite_only or config.active_profile.lower() in {"lite", "mini"}

    commands = registry.list_commands(mode=mode)

    if not commands:
        return "No commands available."

    # Group by type
    by_type = {}
    for cmd in commands:
        if is_lite and cmd["name"] not in registry._lite_commands:
            continue

        cmd_type = cmd["type"]
        if cmd_type not in by_type:
            by_type[cmd_type] = []
        by_type[cmd_type].append(cmd)

    lines = ["[AVAILABLE COMMANDS]"]

    for cmd_type, cmds in sorted(by_type.items()):
        lines.append(f"\n**{cmd_type.upper()}**")
        for cmd in cmds:
            alias_str = f" | {', '.join(cmd['aliases'])}" if cmd['aliases'] else ""
            lines.append(f"  - {cmd['name']}{alias_str}: {cmd['description']}")

    return "\n".join(lines)
```

```python
# -*- coding: utf-8 -*-
"""
Sage Speech-to-Text Module
Handles voice capture and transcription via Faster Whisper sidecar.
"""

import json
import logging
import subprocess
import tempfile
from pathlib import Path
from typing import Dict, Optional, Any
from dataclasses import dataclass

from ..core.state import state
from ..core.config import get_config

logger = logging.getLogger(__name__)


@dataclass
class TranscriptionResult:
    """Result of a speech transcription."""
    ok: bool
    text: str = ""
    language: str = ""
    error: str = ""
    duration_seconds: float = 0.0


class SpeechToText:
    """
    Manages speech-to-text transcription using Faster Whisper sidecar.
    """

    def __init__(self):
        self.config = get_config()
        self.sidecar_python: Optional[Path] = None
        self.sidecar_runner: Optional[Path] = None
        self._sidecar_proc: Optional[subprocess.Popen] = None
        self._sidecar_lock = None
        self._initialize_paths()

    def _initialize_paths(self):
        """Initialize paths to voice sidecar scripts."""
        workspace = Path(__file__).parent.parent.parent
        self.sidecar_python = workspace / "VOICE" / "faster_whisper" / ".venv" / "Scripts" / "python.exe"
        self.sidecar_runner = workspace / "VOICE" / "faster_whisper" / "run_stt.py"

    def is_available(self) -> bool:
        """Check if STT sidecar is available."""
        return self.sidecar_python.exists() and self.sidecar_runner.exists()

    def check_setup(self) -> Dict[str, Any]:
        """Check STT sidecar setup and return status."""
        result = {
            "available": self.is_available(),
            "python_path": str(self.sidecar_python),
            "runner_path": str(self.sidecar_runner),
            "input_devices": 0,
            "default_input": "unknown",
            "error": None,
        }

        if not result["available"]:
            result["error"] = "Sidecar files missing"
            return result

        # Run sidecar check
        try:
            voice_config = self.config.voice
            cmd = [
                str(self.sidecar_python),
                str(self.sidecar_runner),
                "--check",
            ]

            if voice_config.input_device:
                cmd.extend(["--input-device", voice_config.input_device])

            proc = subprocess.run(
                cmd,
                capture_output=True,
                text=True,
                timeout=25,
                cwd=str(Path.cwd()),
            )

            if proc.returncode != 0:
                result["error"] = f"Check failed: {proc.stderr[:260]}"
                return result

            payload = json.loads((proc.stdout or "").strip() or "{}")
            result["input_devices"] = int(payload.get("input_devices", 0))
            result["default_input"] = str(payload.get("default_input", "unknown"))
            result["ok"] = payload.get("ok", False)

            if not result["ok"]:
                result["error"] = payload.get("error", "Unknown error")

        except Exception as e:
            result["error"] = str(e)

        return result

    def capture_once(
        self,
        record_seconds: int = 12,
        model_size: Optional[str] = None,
        language: Optional[str] = None,
        timeout_seconds: Optional[int] = None,
        samplerate: Optional[int] = None,
        silence_reset_seconds: Optional[float] = None,
        stop_on_silence: bool = False,
    ) -> TranscriptionResult:
        """
        Capture and transcribe a single voice recording.

        Args:
            record_seconds: Duration to record in seconds
            model_size: Whisper model size (tiny, base, small, medium, large)
            language: Language code (e.g., "en")
            timeout_seconds: Request timeout
            samplerate: Audio sample rate
            silence_reset_seconds: Silence detection threshold
            stop_on_silence: Stop recording on silence detection

        Returns:
            TranscriptionResult with transcribed text or error
        """
        config = self.config.voice

        if not self.is_available():
            return TranscriptionResult(
                ok=False,
                error="Voice sidecar not set up. Run voice-check for details.",
            )

        # Apply defaults from config
        record_seconds = max(1, min(int(record_seconds or config.record_seconds), 45))
        samplerate = int(samplerate or config.samplerate or 16000)
        model_size = str(model_size or config.model_size or "base").strip() or "base"
        language = str(language or config.language or "en").strip()
        timeout_seconds = max(5, int(timeout_seconds or config.timeout_seconds or 120))

        try:
            silence_reset = float(silence_reset_seconds or config.wake_silence_reset_seconds or 4.0)
        except Exception:
            silence_reset = 4.0

        silence_reset = max(0.5, min(float(silence_reset), 10.0))

        try:
            min_stop = float(config.get("loop_min_stop_seconds", config.wake_min_stop_seconds, 5.5))
        except Exception:
            min_stop = 5.5

        min_stop = max(0.0, min(float(min_stop), float(record_seconds)))

        # Build command
        cmd = [
            str(self.sidecar_python),
            str(self.sidecar_runner),
            "--seconds", str(record_seconds),
            "--samplerate", str(samplerate),
            "--model-size", model_size,
            "--silence-reset-seconds", str(silence_reset),
            "--min-stop-seconds", str(min_stop),
        ]

        if stop_on_silence:
            cmd.append("--stop-on-silence")

        if language and language.lower() != "auto":
            cmd.extend(["--language", language])

        if config.input_device:
            cmd.extend(["--input-device", config.input_device])

        try:
            logger.debug(f"Running STT command: {' '.join(cmd)}")

            proc = subprocess.run(
                cmd,
                capture_output=True,
                text=True,
                timeout=timeout_seconds,
                cwd=str(Path.cwd()),
            )

            if proc.returncode != 0:
                stderr = (proc.stderr or "").strip()
                return TranscriptionResult(
                    ok=False,
                    error=f"Voice sidecar failed: {stderr[:300]}",
                )

            payload = json.loads((proc.stdout or "").strip() or "{}")

            if not payload.get("ok"):
                return TranscriptionResult(
                    ok=False,
                    error=payload.get("error", "No speech detected."),
                )

            return TranscriptionResult(
                ok=True,
                text=str(payload.get("text") or "").strip(),
                language=str(payload.get("language") or ""),
                duration_seconds=record_seconds,
            )

        except subprocess.TimeoutExpired:
            return TranscriptionResult(
                ok=False,
                error="Voice transcription timed out.",
            )
        except Exception as e:
            logger.error(f"STT capture failed: {e}", exc_info=True)
            return TranscriptionResult(
                ok=False,
                error=f"Voice processing failed: {e}",
            )

    def start_persistent_sidecar(self) -> bool:
        """Start the persistent sidecar server for continuous operation."""
        if not self.is_available():
            return False

        config = self.config.voice

        if not config.persistent_sidecar:
            logger.debug("Persistent sidecar disabled in config")
            return False

        if self._sidecar_proc and self._sidecar_proc.poll() is None:
            logger.debug("Persistent sidecar already running")
            return True

        try:
            self._sidecar_proc = subprocess.Popen(
                [str(self.sidecar_python), str(self.sidecar_runner), "--server"],
                stdin=subprocess.PIPE,
                stdout=subprocess.PIPE,
                stderr=subprocess.PIPE,
                text=True,
                bufsize=1,
                cwd=str(Path.cwd()),
            )

            logger.info("Persistent STT sidecar started")
            return True

        except Exception as e:
            logger.error(f"Failed to start persistent sidecar: {e}")
            self._sidecar_proc = None
            return False

    def stop_persistent_sidecar(self, timeout: float = 2.0) -> bool:
        """Stop the persistent sidecar server."""
        if not self._sidecar_proc:
            return True

        try:
            # Send shutdown command
            if self._sidecar_proc.stdin:
                try:
                    self._sidecar_proc.stdin.write(json.dumps({"op": "shutdown"}) + "\n")
                    self._sidecar_proc.stdin.flush()
                except Exception:
                    pass

            # Wait for graceful shutdown
            try:
                self._sidecar_proc.wait(timeout=timeout)
            except subprocess.TimeoutExpired:
                self._sidecar_proc.kill()
                self._sidecar_proc.wait(timeout=1.0)

            logger.info("Persistent STT sidecar stopped")
            return True

        except Exception as e:
            logger.error(f"Failed to stop persistent sidecar: {e}")
            return False
        finally:
            self._sidecar_proc = None
```

```python
# -*- coding: utf-8 -*-
"""
Sage Text-to-Speech Module
Handles voice synthesis using Kokoro ONNX.
"""

import logging
import tempfile
import wave
import os
from pathlib import Path
from typing import Dict, Optional, Any, List
from dataclasses import dataclass

from ..core.state import state
from ..core.config import get_config
from ..core.utils import _normalize_text_for_tts

logger = logging.getLogger(__name__)


@dataclass
class TTSResult:
    """Result of a TTS synthesis."""
    ok: bool
    spoken: str = ""
    purpose: str = "general"
    chunks: int = 0
    engine: str = ""
    voice: str = ""
    error: str = ""
    skipped: bool = False
    reason: str = ""


class TextToSpeech:
    """
    Manages text-to-speech synthesis using Kokoro ONNX.
    """

    def __init__(self):
        self.config = get_config()
        self._engine = None
        self._import_ok = False
        self._import_detail = ""
        self._check_import()

    def _check_import(self):
        """Check if Kokoro ONNX is available."""
        try:
            import kokoro_onnx
            self._engine = kokoro_onnx
            self._import_ok = True
            self._import_detail = str(Path(kokoro_onnx.__file__).resolve())
            logger.info(f"Kokoro ONNX imported from: {self._import_detail}")
        except Exception as e:
            self._import_ok = False
            self._import_detail = str(e)
            logger.warning(f"Kokoro ONNX not available: {e}")

    def is_ready(self) -> bool:
        """Check if TTS engine is ready to synthesize."""
        if not self._import_ok:
            return False

        config = self.config.voice

        if not config.tts_enabled:
            return False

        # Check model and voices files
        model_path = config.voice.kokoro_model_path
        voices_path = config.voice.kokoro_voices_path

        if model_path and not Path(model_path).exists():
            logger.warning(f"TTS model not found: {model_path}")
            return False

        if voices_path and not Path(voices_path).exists():
            logger.warning(f"TTS voices not found: {voices_path}")
            return False

        return True

    def get_status(self) -> Dict[str, Any]:
        """Get TTS engine status."""
        config = self.config.voice

        return {
            "ready": self.is_ready(),
            "import_ok": self._import_ok,
            "import_detail": self._import_detail,
            "tts_enabled": config.tts_enabled,
            "engine": config.tts_engine,
            "voice": config.kokoro_voice,
            "model_path": config.kokoro_model_path or "(unset)",
            "voices_path": config.kokoro_voices_path or "(unset)",
            "model_exists": Path(config.kokoro_model_path).exists() if config.kokoro_model_path else False,
            "voices_exists": Path(config.kokoro_voices_path).exists() if config.kokoro_voices_path else False,
        }

    def speak(
        self,
        text: str,
        purpose: str = "general",
        voice_id: Optional[str] = None,
        speed: Optional[float] = None,
        volume: Optional[float] = None,
    ) -> TTSResult:
        """
        Synthesize speech from text.

        Args:
            text: Text to synthesize
            purpose: Purpose of speech (general, chat, wake, proactive, etc.)
            voice_id: Override voice ID
            speed: Override speed (0.5 - 2.0)
            volume: Override volume (0.0 - 2.0)

        Returns:
            TTSResult with synthesis outcome
        """
        config = self.config.voice

        # Check if TTS is enabled
        if not config.tts_enabled:
            return TTSResult(
                ok=False,
                skipped=True,
                reason="tts_disabled",
                purpose=purpose,
            )

        # Windows only for now
        if os.name != "nt":
            return TTSResult(
                ok=False,
                skipped=True,
                reason="windows_only",
                purpose=purpose,
            )

        # Check engine
        if config.tts_engine != "kokoro":
            return TTSResult(
                ok=False,
                skipped=True,
                reason=f"unsupported_tts_engine:{config.tts_engine}",
                purpose=purpose,
            )

        # Check readiness
        if not self.is_ready():
            return TTSResult(
                ok=False,
                error="kokoro_not_ready",
                purpose=purpose,
            )

        # Normalize text for TTS
        spoken = _normalize_text_for_tts(text)
        if not spoken:
            return TTSResult(
                ok=False,
                skipped=True,
                reason="empty_text",
                purpose=purpose,
            )

        # Apply overrides
        voice_id = voice_id or config.kokoro_voice
        speed = speed if speed is not None else config.kokoro_speed
        volume = volume if volume is not None else config.kokoro_volume

        # Clamp values
        speed = max(0.5, min(2.0, speed))
        volume = max(0.0, min(2.0, volume))

        # Split into chunks for long text
        chunk_limit = self._get_chunk_limit(purpose, spoken)
        chunks = self._split_chunks(spoken, chunk_limit)

        if not chunks:
            return TTSResult(
                ok=False,
                skipped=True,
                reason="empty_text",
                purpose=purpose,
            )

        # Synthesize and play
        try:
            return self._synthesize_and_play(chunks, voice_id, speed, volume, purpose)
        except Exception as e:
            logger.error(f"TTS synthesis failed: {e}", exc_info=True)
            return TTSResult(
                ok=False,
                error=f"kokoro_speak_failed:{e}",
                purpose=purpose,
            )

    def _get_chunk_limit(self, purpose: str, text: str) -> int:
        """Determine chunk size limit based on purpose."""
        config = self.config.voice

        max_chars = max(40, int(config.tts_max_chars or 420))
        default_limit = 220
        configured = max(40, min(int(config.tts_chunk_max_chars or default_limit), 320))

        normalized_purpose = purpose.strip().lower()
        text_len = len(text)

        # Long manual reads benefit from larger chunks
        if normalized_purpose in {"manual", "voice-say"} and text_len >= 650:
            return max(configured, 240)

        return configured

    def _split_chunks(self, text: str, max_chars: int) -> List[str]:
        """Split text into TTS-friendly chunks."""
        value = text.strip()
        if not value:
            return []

        limit = max(40, int(max_chars or 420))

        # Split on sentence boundaries
        sentences = [s.strip() for s in re.split(r"(?<=[.!?])\s+", value) if s.strip()]

        chunks: List[str] = []
        current = ""

        for sentence in sentences:
            if not sentence:
                continue

            combined = (current + " " + sentence).strip() if current else sentence

            if len(combined) <= limit:
                current = combined
            else:
                if current:
                    chunks.append(current)

                # Handle very long sentences
                if len(sentence) > limit:
                    words = sentence.split()
                    long_current = ""
                    for word in words:
                        candidate = (long_current + " " + word).strip() if long_current else word
                        if len(candidate) <= limit:
                            long_current = candidate
                        else:
                            if long_current:
                                chunks.append(long_current)
                            long_current = word[:limit]
                    if long_current:
                        chunks.append(long_current)
                else:
                    current = sentence

        if current:
            chunks.append(current)

        return chunks

    def _synthesize_and_play(
        self,
        chunks: List[str],
        voice_id: str,
        speed: float,
        volume: float,
        purpose: str,
    ) -> TTSResult:
        """Synthesize chunks and play audio."""
        import numpy as np
        import winsound

        Kokoro = getattr(self._engine, "Kokoro")

        config = self.config.voice

        model_path = config.kokoro_model_path
        voices_path = config.kokoro_voices_path
        lang = config.kokoro_lang or "en-us"

        # Initialize engine
        engine = Kokoro(model_path=model_path, voices_path=voices_path)

        pcm_parts = []
        sample_rate = 24000

        for chunk in chunks:
            try:
                audio, chunk_sample_rate = engine.create(
                    chunk,
                    voice=voice_id,
                    speed=speed,
                    lang=lang,
                )

                sample_rate = int(chunk_sample_rate or sample_rate)

                # Convert to PCM
                audio_np = np.asarray(audio, dtype=np.float32)
                audio_np = audio_np * float(volume)
                audio_np = np.clip(audio_np, -1.0, 1.0)
                pcm = (audio_np * 32767.0).astype(np.int16)
                pcm_parts.append(pcm)

            except Exception as e:
                logger.error(f"Failed to synthesize chunk: {e}")
                continue

        if not pcm_parts:
            return TTSResult(
                ok=False,
                skipped=True,
                reason="empty_audio",
                purpose=purpose,
            )

        # Merge all chunks
        merged_pcm = np.concatenate(pcm_parts)

        # Add lead-in silence
        try:
            lead_in_ms = int(config.tts_lead_in_ms or 120)
        except Exception:
            lead_in_ms = 120

        lead_in_ms = max(0, min(300, lead_in_ms))

        if lead_in_ms > 0:
            lead_in_samples = max(1, int((float(lead_in_ms) / 1000.0) * float(sample_rate)))
            lead_in = np.zeros(lead_in_samples, dtype=np.int16)
            merged_pcm = np.concatenate([lead_in, merged_pcm])

        # Play audio
        temp_path = None
        try:
            with tempfile.NamedTemporaryFile(delete=False, suffix=".wav") as tmp:
                temp_path = tmp.name

            with wave.open(temp_path, "wb") as wav_file:
                wav_file.setnchannels(1)
                wav_file.setsampwidth(2)
                wav_file.setframerate(int(sample_rate))
                wav_file.writeframes(merged_pcm.tobytes())

            winsound.PlaySound(temp_path, winsound.SND_FILENAME)

            return TTSResult(
                ok=True,
                spoken=" ".join(chunks),
                purpose=purpose,
                chunks=len(chunks),
                engine="kokoro",
                voice=voice_id,
            )

        except Exception as e:
            logger.error(f"Failed to play audio: {e}")
            return TTSResult(
                ok=False,
                error=f"playback_failed:{e}",
                purpose=purpose,
            )
        finally:
            # Cleanup temp file
            if temp_path and os.path.exists(temp_path):
                try:
                    os.remove(temp_path)
                except Exception:
                    pass

    def stop_playback(self):
        """Stop any active TTS playback."""
        if os.name != "nt":
            return

        try:
            import winsound
            winsound.PlaySound(None, winsound.SND_PURGE)
        except Exception:
            pass
```

```python
# -*- coding: utf-8 -*-
"""
Sage Wake Word Detection Module
Handles wake word listening and trigger detection.
"""

import logging
import threading
import time
import re
from datetime import datetime
from typing import Optional, Callable, Dict, Any
from dataclasses import dataclass

from ..core.state import state
from ..core.config import get_config
from .stt import SpeechToText, TranscriptionResult

logger = logging.getLogger(__name__)


@dataclass
class WakeConfig:
    """Wake word detection configuration."""
    enabled: bool = True
    wake_word: str = "sage"
    aliases: list = None
    fuzzy_enabled: bool = True
    fuzzy_threshold: float = 0.84
    require_prefix: bool = False
    min_chars: int = 3
    debounce_seconds: int = 8
    refractory_seconds: int = 2
    capture_seconds: int = 12
    long_trigger_word: str = "listen"
    long_capture_seconds: int = 20
    ack_tts: bool = False
    ack_console: bool = True
    ack_chime: bool = False
    stop_after_interaction: bool = False

    def __post_init__(self):
        if self.aliases is None:
            self.aliases = []


class WakeListener:
    """
    Manages wake word detection and voice command triggering.
    """

    def __init__(self):
        self.config = get_config()
        self.stt = SpeechToText()
        self._thread: Optional[threading.Thread] = None
        self._stop_event: Optional[threading.Event] = None
        self._running = False
        self._last_trigger_ts = 0.0
        self._callback: Optional[Callable[[str, str, str], None]] = None

    def _get_wake_config(self) -> WakeConfig:
        """Extract wake config from main config."""
        voice = self.config.voice

        return WakeConfig(
            enabled=voice.wake_enabled,
            wake_word=voice.wake_word,
            aliases=voice.wake_aliases,
            fuzzy_enabled=voice.wake_fuzzy_enabled,
            fuzzy_threshold=voice.wake_fuzzy_threshold,
            require_prefix=voice.wake_require_prefix,
            min_chars=voice.wake_min_chars,
            debounce_seconds=voice.wake_debounce_seconds,
            refractory_seconds=voice.wake_refractory_seconds,
            capture_seconds=voice.wake_capture_seconds,
            long_trigger_word=voice.wake_long_trigger_word,
            long_capture_seconds=voice.wake_long_capture_seconds,
            ack_tts=voice.wake_ack_tts,
            ack_console=voice.wake_ack_console,
            ack_chime=voice.wake_ack_chime,
            stop_after_interaction=voice.wake_stop_after_interaction,
        )

    def set_callback(self, callback: Callable[[str, str, str], None]):
        """
        Set the callback for wake word triggers.

        Args:
            callback: Function(transcript, language, wake_word) to call on trigger
        """
        self._callback = callback

    def is_running(self) -> bool:
        """Check if wake listener is running."""
        return self._running and self._thread and self._thread.is_alive()

    def start(self) -> str:
        """Start the wake listener background thread."""
        wake_config = self._get_wake_config()

        if not wake_config.enabled:
            return "[BLOCKED] Wake-word listener disabled in config."

        if not self.stt.is_available():
            return "[WARNING] Voice sidecar not set up. Run voice-check for details."

        if self.is_running():
            return "[CHECK] Wake-word listener already running."

        # Create stop event
        self._stop_event = threading.Event()

        # Start thread
        self._thread = threading.Thread(
            target=self._listen_loop,
            args=(self._stop_event,),
            daemon=True,
            name="sage-wake-listener",
        )
        self._thread.start()

        self._running = True
        state.wake_listener_started = True
        state.set_wake_listener_thread(self._thread, self._stop_event)

        logger.info("Wake-word listener started")
        return "[CHECK] Wake-word listener started. Say 'Sage' to trigger capture."

    def stop(self, timeout: float = 2.0) -> str:
        """Stop the wake listener."""
        if not self.is_running():
            self._running = False
            state.wake_listener_started = False
            return "[CHECK] Wake-word listener already stopped."

        try:
            if self._stop_event:
                self._stop_event.set()

            if self._thread:
                self._thread.join(timeout=timeout)

            logger.info("Wake-word listener stopped")
            return "[CHECK] Wake-word listener stopped."

        except Exception as e:
            logger.error(f"Failed to stop wake listener: {e}")
            return f"[WARNING] Wake stop failed: {e}"
        finally:
            self._running = False
            state.wake_listener_started = False
            state.set_wake_listener_thread(None, None)

    def _listen_loop(self, stop_event: threading.Event):
        """Main wake listener loop."""
        wake_config = self._get_wake_config()

        last_trigger_ts = 0.0
        empty_streak = 0

        logger.info("Wake listener loop started")

        while not stop_event.is_set():
            try:
                # Capture audio
                result = self.stt.capture_once(
                    record_seconds=wake_config.capture_seconds,
                    model_size=self.config.voice.wake_model_size,
                    language=self.config.voice.language,
                    timeout_seconds=self.config.voice.wake_timeout_seconds,
                    samplerate=self.config.voice.samplerate,
                )

                if not result.ok:
                    logger.debug(f"Wake capture error: {result.error}")
                    empty_streak += 1
                    if empty_streak == 1 or empty_streak % 5 == 0:
                        logger.debug(f"Empty transcript streak: {empty_streak}")
                    time.sleep(self.config.voice.wake_idle_seconds)
                    continue

                empty_streak = 0
                transcript = result.text.strip().lower()

                # Check minimum length
                if len(re.sub(r"\s+", "", transcript)) < wake_config.min_chars:
                    logger.debug(f"Too short: {transcript}")
                    time.sleep(self.config.voice.wake_idle_seconds)
                    continue

                # Check for wake word
                detected, matched_word = self._wake_detected(
                    transcript,
                    wake_config.wake_word,
                    wake_config.aliases,
                    wake_config.fuzzy_enabled,
                    wake_config.fuzzy_threshold,
                    wake_config.require_prefix,
                )

                if not detected:
                    logger.debug(f"Wake not detected: {transcript}")
                    time.sleep(self.config.voice.wake_idle_seconds)
                    continue

                # Check debounce
                now_ts = time.time()
                if now_ts - last_trigger_ts < wake_config.debounce_seconds:
                    logger.debug("Debounced")
                    time.sleep(1)
                    continue

                last_trigger_ts = now_ts
                state.set_wake_last_trigger(datetime.now())

                # Acknowledge
                if wake_config.ack_tts:
                    from .tts import TextToSpeech
                    tts = TextToSpeech()
                    tts.speak("Yes, I'm listening.", purpose="wake")

                if wake_config.ack_console:
                    if wake_config.ack_chime:
                        try:
                            print("\a", end="")
                        except Exception:
                            pass
                    logger.info(f"[WAKE ACK] Listening... ({datetime.now().strftime('%Y-%m-%d %H:%M:%S')})")

                # Check for long capture trigger
                use_long_capture = self._uses_long_capture(
                    transcript,
                    matched_word or wake_config.wake_word,
                    wake_config.long_trigger_word,
                )

                capture_seconds = (
                    wake_config.long_capture_seconds
                    if use_long_capture
                    else wake_config.capture_seconds
                )

                # Check for inline command
                inline_payload = self._extract_inline_payload(
                    transcript,
                    matched_word or wake_config.wake_word,
                    wake_config.require_prefix,
                )

                if inline_payload and self._is_meaningful(inline_payload, wake_config.min_chars):
                    logger.debug(f"Inline payload: {inline_payload}")

                    if self._callback:
                        try:
                            self._callback(
                                inline_payload,
                                self.config.voice.language or "en",
                                matched_word or wake_config.wake_word,
                            )
                        except Exception as e:
                            logger.error(f"Wake callback failed: {e}")

                    if wake_config.stop_after_interaction:
                        logger.info("Stopping after interaction (single-interaction mode)")
                        break

                    if wake_config.refractory_seconds > 0:
                        time.sleep(wake_config.refractory_seconds)
                    continue

                # Fallback capture
                logger.debug("Fallback capture")

                capture = self.stt.capture_once(
                    record_seconds=capture_seconds,
                    model_size=self.config.voice.model_size,
                    language=self.config.voice.language,
                    timeout_seconds=self.config.voice.timeout_seconds,
                    samplerate=self.config.voice.samplerate,
                )

                if not capture.ok:
                    logger.warning(f"Wake fallback failed: {capture.error}")
                    time.sleep(wake_config.refractory_seconds)
                    continue

                captured_text = capture.text.strip()
                captured_lang = capture.language or self.config.voice.language or "en"

                if not captured_text:
                    logger.debug("Fallback no speech")
                    time.sleep(wake_config.refractory_seconds)
                    continue

                # Trigger callback
                if self._callback:
                    try:
                        self._callback(captured_text, captured_lang, matched_word or wake_config.wake_word)
                    except Exception as e:
                        logger.error(f"Wake callback failed: {e}")

                if wake_config.stop_after_interaction:
                    logger.info("Stopping after interaction (single-interaction mode)")
                    break

                if wake_config.refractory_seconds > 0:
                    time.sleep(wake_config.refractory_seconds)

            except subprocess.TimeoutExpired:
                time.sleep(self.config.voice.wake_idle_seconds)
            except Exception as e:
                logger.error(f"Wake listener error: {e}", exc_info=True)
                time.sleep(self.config.voice.wake_idle_seconds)

        logger.info("Wake listener loop ended")

    def _wake_detected(
        self,
        transcript: str,
        wake_word: str,
        aliases: list,
        fuzzy_enabled: bool,
        fuzzy_threshold: float,
        require_prefix: bool,
    ) -> tuple:
        """Detect if wake word was spoken."""
        from difflib import SequenceMatcher

        text = transcript.strip().lower()

        # Build candidate list
        candidates = [wake_word.lower()]
        for alias in (aliases or []):
            alias_clean = re.sub(r"[^a-z0-9]", "", str(alias).lower())
            if alias_clean and alias_clean not in candidates:
                candidates.append(alias_clean)

        if not text or not candidates:
            return False, wake_word.lower()

        words = re.findall(r"[a-z0-9']+", text)
        if not words:
            return False, wake_word.lower()

        prefixes = {"hey", "ok", "okay"}

        if require_prefix:
            token = words[0]
            if token in prefixes and len(words) > 1:
                token = words[1]

            for cand in candidates:
                if self._token_match(token, cand, fuzzy_enabled, fuzzy_threshold):
                    return True, cand

            return False, wake_word.lower()

        for token in words:
            for cand in candidates:
                if self._token_match(token, cand, fuzzy_enabled, fuzzy_threshold):
                    return True, cand

        return False, wake_word.lower()

    def _token_match(
        self,
        token: str,
        candidate: str,
        fuzzy_enabled: bool,
        threshold: float,
    ) -> bool:
        """Check if token matches candidate."""
        from difflib import SequenceMatcher

        t = re.sub(r"[^a-z0-9]", "", str(token).lower())
        c = re.sub(r"[^a-z0-9]", "", str(candidate).lower())

        if not t or not c:
            return False

        if t == c:
            return True

        if not fuzzy_enabled:
            return False

        if abs(len(t) - len(c)) > 2:
            return False

        if t[:1] != c[:1]:
            return False

        score = SequenceMatcher(None, t, c).ratio()
        return score >= threshold

    def _uses_long_capture(self, transcript: str, wake_word: str, trigger_word: str) -> bool:
        """Check if long capture mode should be used."""
        text = transcript.strip().lower()
        wake = wake_word.lower()
        trigger = trigger_word.lower()

        if not text or not wake or not trigger:
            return False

        normalized_text = re.sub(r"[^a-z0-9]+", " ", text).strip()
        normalized_wake = re.sub(r"[^a-z0-9]+", " ", wake).strip()
        normalized_trigger = re.sub(r"[^a-z0-9]+", " ", trigger).strip()

        pattern = rf"^(?:(?:hey|ok|okay)\s+)?{re.escape(normalized_wake)}\s+{re.escape(normalized_trigger)}\b"
        return bool(re.search(pattern, normalized_text))

    def _extract_inline_payload(self, transcript: str, wake_word: str, require_prefix: bool) -> str:
        """Extract command text spoken with wake word."""
        text = transcript.strip()

        if not text:
            return ""

        wake = wake_word.lower()

        if not wake:
            return ""

        lower_text = text.lower()

        if require_prefix:
            m = re.match(rf"^(?:(?:hey|ok|okay)\s+)?{re.escape(wake)}\b[\s,;:!?-]*(.*)$", lower_text)
            if not m:
                return ""
            payload = (m.group(1) or "").strip()
            payload = re.sub(r"^[^a-z0-9]+", "", payload)
            payload = re.sub(r"\s+", " ", payload).strip()
            return payload

        idx = lower_text.find(wake)
        if idx < 0:
            return ""

        tail = lower_text[idx + len(wake):]
        tail = re.sub(r"^[\s,;:!?-]+", "", tail)
        tail = re.sub(r"^[^a-z0-9]+", "", tail)
        tail = re.sub(r"\s+", " ", tail).strip()
        return tail

    def _is_meaningful(self, text: str, min_chars: int) -> bool:
        """Check if text is meaningful."""
        value = str(text or "").strip()
        if not value:
            return False

        compact = re.sub(r"\s+", "", value)
        if len(compact) < max(1, int(min_chars or 1)):
            return False

        return bool(re.search(r"[A-Za-z0-9]", value))

    def get_status(self) -> Dict[str, Any]:
        """Get wake listener status."""
        wake_config = self._get_wake_config()

        return {
            "enabled": wake_config.enabled,
            "running": self.is_running(),
            "wake_word": wake_config.wake_word,
            "model_size": self.config.voice.wake_model_size,
            "capture_seconds": wake_config.capture_seconds,
            "long_trigger": f"{wake_config.wake_word}-{wake_config.long_trigger_word}",
            "long_capture_seconds": wake_config.long_capture_seconds,
            "debug": f"on({wake_config.wake_debug_mode})" if wake_config.wake_debug else "off",
            "stop_after": "on" if wake_config.stop_after_interaction else "off",
            "last_trigger": state.get_wake_last_trigger().strftime("%Y-%m-%d %H:%M:%S") if state.get_wake_last_trigger() else "never",
        }
```

```python
# -*- coding: utf-8 -*-
"""
Sage Voice Conversation Loop Module
Handles continuous voice conversation sessions.
"""

import logging
import threading
import time
import re
from datetime import datetime
from typing import Optional, List, Callable
from dataclasses import dataclass

from ..core.state import state
from ..core.config import get_config
from .stt import SpeechToText
from .tts import TextToSpeech

logger = logging.getLogger(__name__)


@dataclass
class VoiceLoopConfig:
    """Voice loop configuration."""
    record_seconds: int = 12
    max_turns: int = 8
    silence_limit: int = 2
    stop_on_silence: bool = False
    silence_reset_seconds: float = 4.0
    min_stop_seconds: float = 5.5
    stop_phrases: List[str] = None

    def __post_init__(self):
        if self.stop_phrases is None:
            self.stop_phrases = [
                "stop listening",
                "stop voice",
                "stop conversation",
                "voice off",
            ]


class VoiceLoop:
    """
    Manages continuous voice conversation sessions.
    """

    def __init__(self):
        self.config = get_config()
        self.stt = SpeechToText()
        self.tts = TextToSpeech()
        self._session_id: int = 0
        self._stop_event: Optional[threading.Event] = None

    def _get_loop_config(self) -> VoiceLoopConfig:
        """Extract loop config from main config."""
        voice = self.config.voice

        return VoiceLoopConfig(
            record_seconds=voice.record_seconds,
            max_turns=voice.loop_max_turns,
            silence_limit=voice.loop_silence_limit,
            stop_on_silence=voice.loop_stop_on_silence,
            silence_reset_seconds=voice.loop_silence_reset_seconds,
            min_stop_seconds=voice.loop_min_stop_seconds,
            stop_phrases=voice.stop_phrases,
        )

    def begin_session(self) -> int:
        """Start a new voice loop session."""
        session_id = state.voice_loop_begin_session()
        self._session_id = session_id
        self._stop_event = state.get_voice_loop_stop_event()
        logger.info(f"Voice loop session started: {session_id}")
        return session_id

    def cancel_session(self) -> int:
        """Cancel the active voice loop session."""
        session_id = state.voice_loop_cancel_session()
        self._session_id = 0
        self._stop_event = None
        logger.info(f"Voice loop session cancelled: {session_id}")
        return session_id

    def is_active(self, session_id: int) -> bool:
        """Check if a session is active."""
        return state.voice_loop_is_active(session_id)

    def run(
        self,
        route_override: Optional[str] = None,
        emit_live: bool = False,
        tts_voice_override: Optional[str] = None,
        process_transcript: Callable[[str], str] = None,
    ) -> str:
        """
        Run a voice conversation loop.

        Args:
            route_override: Force routing to specific model (deep, deep-local)
            emit_live: Emit transcripts live via wake callback
            tts_voice_override: Override TTS voice
            process_transcript: Function to process transcript and get response

        Returns:
            Final status message
        """
        loop_config = self._get_loop_config()
        voice_config = self.config.voice

        session_id = self.begin_session()

        silence_count = 0
        lines: List[str] = []
        tts_notice_emitted = False

        logger.info(f"Voice loop running (session={session_id}, turns={loop_config.max_turns})")

        for turn in range(1, loop_config.max_turns + 1):
            # Check session active
            if not self.is_active(session_id):
                logger.debug("Session no longer active")
                break

            # Check stop event
            if self._stop_event and self._stop_event.is_set():
                lines.append("[VOICE LOOP] Stop requested.")
                if emit_live:
                    state.trigger_wake_event("[VOICE LOOP] Stop requested.")
                break

            try:
                # Capture
                result = self.stt.capture_once(
                    record_seconds=loop_config.record_seconds,
                    silence_reset_seconds=loop_config.silence_reset_seconds,
                    stop_on_silence=loop_config.stop_on_silence,
                    model_size=voice_config.model_size,
                    language=voice_config.language,
                    timeout_seconds=voice_config.timeout_seconds,
                    samplerate=voice_config.samplerate,
                )

                # Check session again (may have changed during capture)
                if not self.is_active(session_id):
                    break

                if not result.ok:
                    err_text = result.error.lower()

                    if "no speech detected" in err_text:
                        silence_count += 1
                        msg = f"[VOICE LOOP] Turn {turn}: no speech ({silence_count}/{loop_config.silence_limit})."
                        lines.append(msg)

                        if emit_live:
                            state.trigger_wake_event(msg)

                        if silence_count >= loop_config.silence_limit:
                            lines.append("[VOICE LOOP] Stopped due to silence.")
                            break

                        continue

                    msg = f"[WARNING] Turn {turn}: {result.error}"
                    lines.append(msg)

                    if emit_live:
                        state.trigger_wake_event(msg)

                    break

                # Process transcript
                transcript = result.text.strip()
                detected_lang = result.language or voice_config.language or "en"

                if not transcript:
                    silence_count += 1
                    lines.append(f"[VOICE LOOP] Turn {turn}: no speech ({silence_count}/{loop_config.silence_limit}).")

                    if silence_count >= loop_config.silence_limit:
                        lines.append("[VOICE LOOP] Stopped due to silence.")
                        break

                    continue

                silence_count = 0

                # Log transcript
                transcript_row = f"[VOICE TRANSCRIPT {turn}] ({detected_lang}) {transcript}"
                lines.append(transcript_row)

                if emit_live:
                    state.trigger_wake_event(transcript_row)

                # Check stop phrases
                transcript_lower = transcript.lower()
                if any(p in transcript_lower for p in (loop_config.stop_phrases or [])):
                    stop_row = "[VOICE LOOP] Stop phrase detected."
                    lines.append(stop_row)

                    if emit_live:
                        state.trigger_wake_event(stop_row)

                    break

                # Process through callback
                if process_transcript:
                    try:
                        response = process_transcript(transcript)
                        response_row = self._sanitize_response(response)
                        lines.append(response_row)

                        if emit_live:
                            state.trigger_wake_event(response_row)

                        # Speak response
                        self._speak_response(response_row, tts_voice_override, voice_config, route_override)

                    except Exception as e:
                        logger.error(f"Transcript processing failed: {e}")
                        lines.append(f"[WARNING] Turn {turn}: processing failed: {e}")

            except Exception as e:
                logger.error(f"Voice loop turn failed: {e}", exc_info=True)
                lines.append(f"[WARNING] Turn {turn}: {e}")
                break

        else:
            max_row = "[VOICE LOOP] Max turns reached."
            lines.append(max_row)

            if emit_live:
                state.trigger_wake_event(max_row)

        self.cancel_session()

        if emit_live:
            return "[VOICE LOOP LIVE] completed"

        return "\n\n".join(lines)

    def _sanitize_response(self, text: str) -> str:
        """Sanitize response for voice loop."""
        import re

        raw = str(text or "").strip()
        if not raw:
            return ""

        # Remove LaTeX markers
        raw = re.sub(r"\$\$(.+?)\$\$", r"\1", raw, flags=re.DOTALL)
        raw = re.sub(r"\$(.+?)\$", r"\1", raw, flags=re.DOTALL)

        # Strip backend artifacts
        raw = re.sub(r"<\|[^\n>]+\|>", " ", raw)
        raw = re.sub(r"</?think[^>]*>", " ", raw, flags=re.IGNORECASE)
        raw = re.sub(r"\s+", " ", raw).strip()

        return raw

    def _speak_response(
        self,
        response: str,
        voice_override: Optional[str],
        voice_config,
        route_override: Optional[str],
    ):
        """Speak a response using TTS."""
        if not voice_config.tts_enabled:
            return

        spoken = self._sanitize_response(response)

        if not spoken:
            return

        preferred_voice = voice_override

        if preferred_voice:
            result = self.tts.speak(
                spoken,
                purpose="chat",
                voice_id=preferred_voice,
                speed=voice_config.kokoro_speed,
                volume=voice_config.kokoro_volume,
            )

            if not result.ok:
                self.tts.speak(spoken, purpose="chat")

            return

        if route_override == "deep":
            narrator_voice = voice_config.kokoro_narrator_voice
            result = self.tts.speak(
                spoken,
                purpose="chat",
                voice_id=narrator_voice,
                speed=voice_config.kokoro_narrator_speed,
                volume=voice_config.kokoro_narrator_volume,
            )

            if not result.ok:
                self.tts.speak(spoken, purpose="chat")

            return

        self.tts.speak(spoken, purpose="chat")
```

```python
# -*- coding: utf-8 -*-
"""
Sage File Manager
Handles file and folder operations within the workspace.
"""

import logging
import os
import shutil
from pathlib import Path
from typing import Dict, List, Optional, Any, Tuple
from dataclasses import dataclass

from ..core.state import state
from ..core.config import get_config

logger = logging.getLogger(__name__)


@dataclass
class FileEntry:
    """Represents a file or folder entry."""
    name: str
    path: Path
    is_directory: bool
    size_bytes: int = 0
    modified: Optional[float] = None


class FileManager:
    """
    Manages file and folder operations within the Sage workspace.
    All operations are workspace-scoped for security.
    """

    # Protected files that cannot be modified
    PROTECTED_FILES = {
        "sage.py", "sage_lite.py", "main.py",
        "sage_config.json", "config.json",
        ".gitignore", "file_index.json",
    }

    # Protected extensions
    PROTECTED_EXTS = {".py", ".json", ".yaml", ".yml", ".ini", ".cfg"}

    def __init__(self, workspace_root: Optional[Path] = None):
        self.workspace_root = workspace_root or Path.cwd()
        self.config = get_config()

    def _resolve_path(self, raw_path: str) -> Path:
        """
        Resolve a path and ensure it stays within workspace.

        Args:
            raw_path: User-provided path string

        Returns:
            Resolved Path object

        Raises:
            ValueError: If path is outside workspace
        """
        value = str(raw_path or "").strip().strip('"').strip("'")

        if not value:
            raise ValueError("Path is empty.")

        candidate = Path(value)

        # Resolve absolute vs relative
        if candidate.is_absolute():
            resolved = candidate.resolve()
        else:
            resolved = (self.workspace_root / candidate).resolve()

        # Security check: must be within workspace
        try:
            resolved.relative_to(self.workspace_root)
        except ValueError:
            raise ValueError(f"Path must stay inside workspace: {raw_path}")

        return resolved

    def _is_protected(self, path: Path) -> bool:
        """Check if a path is protected from modification."""
        return (
            path.name in self.PROTECTED_FILES or
            path.suffix.lower() in self.PROTECTED_EXTS
        )

    def list_directory(
        self,
        path: str = ".",
        folders_only: bool = False,
        limit: int = 300,
    ) -> Dict[str, Any]:
        """
        List contents of a directory.

        Args:
            path: Directory path (relative to workspace)
            folders_only: If True, only show folders
            limit: Maximum entries to return

        Returns:
            Dict with entries list and metadata
        """
        try:
            target = self._resolve_path(path)
        except ValueError as e:
            return {"ok": False, "error": str(e)}

        if not target.exists():
            return {
                "ok": False,
                "error": f"Path not found: {target.relative_to(self.workspace_root)}",
            }

        if not target.is_dir():
            return {
                "ok": False,
                "error": f"Not a directory: {target.relative_to(self.workspace_root)}",
            }

        try:
            entries = list(target.iterdir())

            # Sort: folders first, then alphabetically
            entries.sort(key=lambda p: (not p.is_dir(), p.name.lower()))

            # Filter folders only if requested
            if folders_only:
                entries = [p for p in entries if p.is_dir()]

            # Build entry list
            entry_list: List[Dict[str, Any]] = []
            for p in entries[:limit]:
                entry = {
                    "name": p.name,
                    "is_directory": p.is_dir(),
                    "path": str(p.relative_to(self.workspace_root)),
                }

                if p.is_file():
                    try:
                        entry["size_bytes"] = p.stat().st_size
                    except Exception:
                        entry["size_bytes"] = 0

                entry_list.append(entry)

            # Check if truncated
            truncated = len(entries) > limit

            return {
                "ok": True,
                "path": str(target.relative_to(self.workspace_root)),
                "entries": entry_list,
                "total_count": len(entries),
                "truncated": truncated,
                "truncated_count": len(entries) - limit if truncated else 0,
            }

        except PermissionError:
            return {"ok": False, "error": "Permission denied"}
        except Exception as e:
            logger.error(f"List directory failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def create_directory(self, path: str) -> Dict[str, Any]:
        """
        Create a directory (including parents).

        Args:
            path: Directory path to create

        Returns:
            Result dict with success status
        """
        try:
            target = self._resolve_path(path)
        except ValueError as e:
            return {"ok": False, "error": str(e)}

        rel_path = target.relative_to(self.workspace_root)

        # Check if already exists
        if target.exists():
            if target.is_dir():
                return {
                    "ok": True,
                    "message": f"Folder already exists: {rel_path}",
                    "existed": True,
                }
            else:
                return {
                    "ok": False,
                    "error": f"Path exists and is not a folder: {rel_path}",
                }

        # Check protection
        if self._is_protected(target):
            return {
                "ok": False,
                "error": "Protected files/extensions are not allowed for mkdir target.",
            }

        try:
            target.mkdir(parents=True, exist_ok=True)
            logger.info(f"Directory created: {rel_path}")

            return {
                "ok": True,
                "message": f"Folder created: {rel_path}",
                "path": str(rel_path),
            }

        except PermissionError:
            return {"ok": False, "error": "Permission denied"}
        except Exception as e:
            logger.error(f"Create directory failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def create_file(self, path: str) -> Dict[str, Any]:
        """
        Create an empty file (without overwriting).

        Args:
            path: File path to create

        Returns:
            Result dict with success status
        """
        try:
            target = self._resolve_path(path)
        except ValueError as e:
            return {"ok": False, "error": str(e)}

        rel_path = target.relative_to(self.workspace_root)

        # Check protection
        if self._is_protected(target):
            return {
                "ok": False,
                "error": "Protected files/extensions are not allowed for touch.",
            }

        # Check if exists
        if target.exists():
            return {
                "ok": False,
                "error": f"File already exists: {rel_path}",
            }

        # Check parent exists
        parent = target.parent
        if not parent.exists():
            return {
                "ok": False,
                "error": f"Parent folder does not exist: {parent.relative_to(self.workspace_root)}. Use mkdir first.",
            }

        try:
            target.touch(exist_ok=False)
            logger.info(f"File created: {rel_path}")

            return {
                "ok": True,
                "message": f"File created: {rel_path}",
                "path": str(rel_path),
            }

        except PermissionError:
            return {"ok": False, "error": "Permission denied"}
        except Exception as e:
            logger.error(f"Create file failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def move_file(
        self,
        source: str,
        target: str,
    ) -> Dict[str, Any]:
        """
        Move/rename a file or folder (no overwrite).

        Args:
            source: Source path
            target: Target path

        Returns:
            Result dict with success status
        """
        try:
            source_path = self._resolve_path(source)
            target_path = self._resolve_path(target)
        except ValueError as e:
            return {"ok": False, "error": str(e)}

        rel_source = source_path.relative_to(self.workspace_root)
        rel_target = target_path.relative_to(self.workspace_root)

        # Check source exists
        if not source_path.exists():
            return {"ok": False, "error": f"Source path does not exist: {rel_source}"}

        # Check protection
        if self._is_protected(source_path) or self._is_protected(target_path):
            return {
                "ok": False,
                "error": "Protected files/extensions are not allowed for move/rename.",
            }

        # Check same path
        if source_path == target_path:
            return {"ok": True, "message": f"Source and target are the same: {rel_source}"}

        # Check target doesn't exist
        if target_path.exists():
            return {"ok": False, "error": f"Target already exists: {rel_target}"}

        # Check not moving folder into itself
        if source_path.is_dir():
            try:
                target_path.relative_to(source_path)
                return {"ok": False, "error": "Cannot move a folder into itself."}
            except ValueError:
                pass

        # Check target parent exists
        parent = target_path.parent
        if not parent.exists():
            return {
                "ok": False,
                "error": f"Target parent folder does not exist: {parent.relative_to(self.workspace_root)}. Use mkdir first.",
            }

        try:
            os.rename(source_path, target_path)
            logger.info(f"Moved: {rel_source} -> {rel_target}")

            return {
                "ok": True,
                "message": f"Moved: {rel_source} -> {rel_target}",
                "source": str(rel_source),
                "target": str(rel_target),
            }

        except PermissionError:
            return {"ok": False, "error": "Permission denied"}
        except Exception as e:
            logger.error(f"Move file failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def delete_path(
        self,
        path: str,
        confirm: bool = False,
    ) -> Dict[str, Any]:
        """
        Delete a file or folder (requires confirmation).

        Args:
            path: Path to delete
            confirm: If True, proceed with deletion

        Returns:
            Result dict with success status
        """
        from ..core.state import state

        # Handle confirmation
        if confirm:
            pending = state.delete_pending
            if not pending:
                return {"ok": False, "error": "No pending delete action. Use delete [path] first."}

            try:
                pending_path = Path(pending["path"])
                pending_path.relative_to(self.workspace_root)
            except Exception:
                state.delete_pending = None
                return {"ok": False, "error": "Pending delete path is invalid."}

            if not pending_path.exists():
                state.delete_pending = None
                return {"ok": False, "error": f"Target no longer exists: {pending_path.name}"}

            if self._is_protected(pending_path):
                state.delete_pending = None
                return {"ok": False, "error": "Protected files/extensions cannot be deleted."}

            try:
                rel_path = pending_path.relative_to(self.workspace_root)

                if pending_path.is_dir():
                    shutil.rmtree(pending_path)
                    msg = f"Folder deleted: {rel_path}"
                else:
                    os.remove(pending_path)
                    msg = f"File deleted: {rel_path}"

                state.delete_pending = None
                logger.info(f"Deleted: {rel_path}")

                return {"ok": True, "message": msg}

            except Exception as e:
                state.delete_pending = None
                logger.error(f"Delete failed: {e}", exc_info=True)
                return {"ok": False, "error": str(e)}

        # Initial delete request - set pending
        try:
            target = self._resolve_path(path)
        except ValueError as e:
            return {"ok": False, "error": str(e)}

        rel_path = target.relative_to(self.workspace_root)

        if not target.exists():
            return {"ok": False, "error": f"Target path does not exist: {rel_path}"}

        if self._is_protected(target):
            return {"ok": False, "error": "Protected files/extensions cannot be deleted."}

        target_type = "folder" if target.is_dir() else "file"

        # Set pending delete in state
        state.delete_pending = {
            "path": str(target),
            "type": target_type,
            "rel_path": str(rel_path),
        }

        return {
            "ok": True,
            "pending": True,
            "message": f"Delete {target_type}: {rel_path}",
            "instruction": "Type 'delete confirm' to proceed or 'delete cancel' to stop.",
        }

    def cancel_delete(self) -> Dict[str, Any]:
        """Cancel a pending delete operation."""
        from ..core.state import state

        if state.delete_pending:
            state.delete_pending = None
            return {"ok": True, "message": "Pending delete canceled."}

        return {"ok": True, "message": "No pending delete action to cancel."}

    def get_pending_delete(self) -> Optional[Dict[str, Any]]:
        """Get the pending delete operation info."""
        from ..core.state import state
        return state.delete_pending
```

```python
# -*- coding: utf-8 -*-
"""
Sage Reminder Manager
Handles reminder creation, listing, and management.
"""

import logging
import re
import time
from datetime import datetime, timedelta
from pathlib import Path
from typing import Dict, List, Optional, Any, Tuple
from dataclasses import dataclass

from ..core.state import state
from ..core.config import get_config
from ..core.utils import _validate_hhmm

logger = logging.getLogger(__name__)


@dataclass
class Reminder:
    """Represents a reminder entry."""
    date: str
    time: str
    description: str
    workspace: str = "main"
    tags: List[str] = None

    def __post_init__(self):
        if self.tags is None:
            self.tags = []

    def to_line(self) -> str:
        """Convert to file line format."""
        tags_str = " ".join(f"[{t}]" for t in self.tags) if self.tags else ""
        desc = f"{self.time} {self.description} {tags_str}".strip()
        return f"{self.date} | {desc}"

    @classmethod
    def from_line(cls, line: str, workspace: str = "main") -> Optional['Reminder']:
        """Parse from file line format."""
        line = line.strip()
        if not line or "|" not in line:
            return None

        try:
            date_part, desc = line.split(" | ", 1)
            date_part = date_part.strip()
            desc = desc.strip()

            # Validate date
            datetime.strptime(date_part, "%Y-%m-%d")

            # Extract time
            time_match = re.search(r'(\d{2}:\d{2})', desc)
            time_str = time_match.group(1) if time_match else "08:00"

            # Extract tags
            tags = re.findall(r'\[([^\]]+)\]', desc)

            # Clean description
            clean_desc = re.sub(r'\d{2}:\d{2}\s*', '', desc).strip()
            clean_desc = re.sub(r'\[[^\]]+\]\s*', '', clean_desc).strip()

            return cls(
                date=date_part,
                time=time_str,
                description=clean_desc,
                workspace=workspace,
                tags=tags,
            )

        except Exception:
            return None


class ReminderManager:
    """
    Manages reminders with support for workspaces, tags, and deduplication.
    """

    def __init__(self, workspace_root: Optional[Path] = None):
        self.workspace_root = workspace_root or Path.cwd()
        self.config = get_config()
        self._reminders_file: Optional[Path] = None

    def _get_reminders_file(self, create: bool = False) -> Path:
        """Get the reminders file path."""
        if self._reminders_file is None:
            # Check config for custom path
            storage_cfg = self.config.storage if hasattr(self.config, 'storage') else {}

            if storage_cfg.get("single_store_mode", True):
                self._reminders_file = self.workspace_root / "NOTES" / "reminders.txt"
            else:
                workspace = state.current_workspace
                self._reminders_file = (
                    self.workspace_root / "WORKSPACES" / workspace / "REMINDERS" / "reminders.txt"
                )

        if create and not self._reminders_file.parent.exists():
            self._reminders_file.parent.mkdir(parents=True, exist_ok=True)

        return self._reminders_file

    def _get_default_time(self) -> str:
        """Get default reminder time from config."""
        reminders_cfg = self.config.reminders

        if reminders_cfg.use_fixed_bell_time:
            return reminders_cfg.fixed_bell_time or "08:00"

        # Could use per-reminder time, but need a default
        return "08:00"

    def _normalize_reminder_datetime(
        self,
        text: str,
        date_str: str = "",
        time_str: str = "",
    ) -> Tuple[str, str]:
        """
        Normalize date and time from various input formats.

        Returns:
            Tuple of (date_str, time_str) in YYYY-MM-DD and HH:MM format
        """
        now = datetime.now()
        raw_text = (text or "").strip().lower()
        date_raw = (date_str or "").strip().lower()
        time_raw = (time_str or "").strip().lower()

        date_part = ""
        time_part = ""

        # === Normalize Date ===

        # Relative phrases in text take precedence
        if " tomorrow" in f" {raw_text}":
            date_part = (now + timedelta(days=1)).strftime("%Y-%m-%d")
        elif " today" in f" {raw_text}":
            date_part = now.strftime("%Y-%m-%d")
        elif "next week" in raw_text:
            date_part = (now + timedelta(days=7)).strftime("%Y-%m-%d")
        else:
            # Weekday matching
            weekday_map = {
                "monday": 0, "tuesday": 1, "wednesday": 2,
                "thursday": 3, "friday": 4, "saturday": 5, "sunday": 6,
            }

            weekday_match = re.search(r"\bnext\s+(monday|tuesday|wednesday|thursday|friday|saturday|sunday)\b", raw_text)
            if weekday_match:
                target = weekday_map[weekday_match.group(1)]
                today_wd = now.weekday()
                days_ahead = (target - today_wd) % 7
                if days_ahead == 0:
                    days_ahead = 7
                date_part = (now + timedelta(days=days_ahead)).strftime("%Y-%m-%d")

        # Try parsing explicit date
        if not date_part and re.match(r"^\d{4}-\d{2}-\d{2}$", date_raw):
            try:
                parsed = datetime.strptime(date_raw, "%Y-%m-%d")
                date_part = parsed.strftime("%Y-%m-%d")
            except Exception:
                pass

        # Fallback date resolution
        if not date_part:
            if date_raw == "today":
                date_part = now.strftime("%Y-%m-%d")
            elif date_raw == "tomorrow":
                date_part = (now + timedelta(days=1)).strftime("%Y-%m-%d")
            else:
                # Try to find date in text
                text_date_match = re.search(r"\b(\d{4}-\d{2}-\d{2})\b", raw_text)
                if text_date_match:
                    try:
                        parsed = datetime.strptime(text_date_match.group(1), "%Y-%m-%d")
                        date_part = parsed.strftime("%Y-%m-%d")
                    except Exception:
                        pass

        # Default to today if no date found
        if not date_part:
            date_part = now.strftime("%Y-%m-%d")

        # === Normalize Time ===

        # Try parsing explicit time
        if time_raw:
            validated = _validate_hhmm(time_raw)
            if validated:
                time_part = validated

        # Time-of-day keywords
        if not time_part and time_raw in {"morning", "afternoon", "evening"}:
            time_part = {"morning": "09:00", "afternoon": "14:00", "evening": "19:00"}[time_raw]

        # Search in text
        if not time_part:
            time_match = re.search(r"\bat\s+(\d{1,2})(?::(\d{2}))?\s*(am|pm)?\b", raw_text)
            if time_match:
                hour = int(time_match.group(1))
                minute = int(time_match.group(2) or "0")
                meridian = (time_match.group(3) or "").lower()

                if meridian == "pm" and 1 <= hour <= 11:
                    hour += 12
                elif meridian == "am" and hour == 12:
                    hour = 0

                if 0 <= hour <= 23 and 0 <= minute <= 59:
                    time_part = f"{hour:02d}:{minute:02d}"

        # Default time
        if not time_part:
            time_part = self._get_default_time()

        return date_part, time_part

    def _is_recent_duplicate(self, date: str, time_str: str, desc: str) -> bool:
        """Check if this reminder is a recent duplicate."""
        window = self.config.reminders.voice_duplicate_guard_seconds or 180
        window = max(0, min(window, 3600))

        if window <= 0:
            return False

        # Normalize description for comparison
        normalized_desc = re.sub(r"\[ws:[^\]]+\]", "", desc.lower())
        normalized_desc = re.sub(r"\[[a-z0-9_-]+\]", "", normalized_desc)
        normalized_desc = re.sub(r"\s+", " ", normalized_desc).strip(" .,:;!?-")

        workspace = state.current_workspace
        key = (workspace.lower(), date, time_str, normalized_desc)

        return state.check_recent_reminder(key, window)

    def add_reminder(
        self,
        date: str,
        description: str,
        time: Optional[str] = None,
        tags: Optional[List[str]] = None,
    ) -> Dict[str, Any]:
        """
        Add a new reminder.

        Args:
            date: Date in YYYY-MM-DD format
            description: Reminder description
            time: Time in HH:MM format (optional)
            tags: List of tags (optional)

        Returns:
            Result dict with success status
        """
        # Validate date
        try:
            datetime.strptime(date, "%Y-%m-%d")
        except ValueError:
            return {
                "ok": False,
                "error": f"Invalid date: {date}. Use YYYY-MM-DD format.",
            }

        # Normalize time
        time_str = _validate_hhmm(time) if time else self._get_default_time()

        # Get workspace tags
        workspace = state.current_workspace
        tags = tags or []

        # Add workspace tag
        ws_tag = f"ws:{workspace}"
        if ws_tag not in tags:
            tags.append(ws_tag)
        if workspace not in tags:
            tags.append(workspace)

        # Build description with tags
        tags_str = " ".join(f"[{t}]" for t in tags)
        full_desc = f"{time_str} {description} {tags_str}".strip()

        # Check for duplicate
        if self._is_recent_duplicate(date, time_str, full_desc):
            return {
                "ok": False,
                "skipped": True,
                "reason": "duplicate",
                "message": f"Duplicate reminder ignored: {date} at {time_str}",
            }

        # Check physical safety
        physical_demand_keywords = {"move", "heavy", "boxes", "clean", "lift", "carry", "garage"}
        needs_safety_notice = any(kw in description.lower() for kw in physical_demand_keywords)

        # Write to file
        filepath = self._get_reminders_file(create=True)
        content = f"{date} | {full_desc}\n"

        try:
            with open(filepath, 'a', encoding='utf-8') as f:
                f.write(content)

            logger.info(f"Reminder added: {date} | {full_desc}")

            response = f"[CHECK] Reminder added: {date} at {time_str} - {description}"

            if needs_safety_notice:
                response += "\n[SAFETY] This looks physically demanding. Please watch your energy."

            return {
                "ok": True,
                "message": response,
                "date": date,
                "time": time_str,
                "description": description,
                "tags": tags,
            }

        except Exception as e:
            logger.error(f"Add reminder failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def add_reminder_from_text(
        self,
        text: str,
        date_str: str = "",
        time_str: str = "",
    ) -> Dict[str, Any]:
        """
        Add a reminder from natural language text.

        Args:
            text: Natural language reminder text
            date_str: Optional explicit date override
            time_str: Optional explicit time override

        Returns:
            Result dict with success status
        """
        # Normalize date/time from text
        date_part, time_part = self._normalize_reminder_datetime(text, date_str, time_str)

        # Extract description (remove date/time keywords)
        desc = text
        desc = re.sub(r'\d{4}-\d{2}-\d{2}', '', desc)
        desc = re.sub(r'\d{1,2}:\d{2}', '', desc)
        desc = re.sub(r'\b(today|tomorrow|next week|morning|afternoon|evening)\b', '', desc, flags=re.IGNORECASE)
        desc = re.sub(r'\bat\s+\d{1,2}(?::\d{2})?\s*(am|pm)?\b', '', desc, flags=re.IGNORECASE)
        desc = re.sub(r'\s+', ' ', desc).strip(" .:-")

        if not desc:
            desc = "Reminder"

        return self.add_reminder(date_part, desc, time_part)

    def list_reminders(
        self,
        include_past: bool = False,
        limit: int = 50,
        tag_filter: Optional[str] = None,
    ) -> Dict[str, Any]:
        """
        List pending reminders.

        Args:
            include_past: If True, include past reminders
            limit: Maximum reminders to return
            tag_filter: Filter by tag (optional)

        Returns:
            Dict with reminders list and metadata
        """
        filepath = self._get_reminders_file()

        if not filepath.exists():
            return {
                "ok": True,
                "reminders": [],
                "count": 0,
                "message": "No reminders set.",
            }

        today = datetime.now().strftime("%Y-%m-%d")
        reminders: List[Dict[str, Any]] = []

        try:
            with open(filepath, 'r', encoding='utf-8') as f:
                lines = f.readlines()

            for line in lines:
                reminder = Reminder.from_line(line)

                if reminder is None:
                    continue

                # Filter by date
                if not include_past and reminder.date < today:
                    continue

                # Filter by tag
                if tag_filter:
                    tag_lower = tag_filter.lower()
                    if not any(tag_lower in t.lower() for t in reminder.tags):
                        continue

                reminders.append({
                    "date": reminder.date,
                    "time": reminder.time,
                    "description": reminder.description,
                    "tags": reminder.tags,
                    "workspace": reminder.workspace,
                    "is_today": reminder.date == today,
                    "is_overdue": reminder.date < today,
                })

                if len(reminders) >= limit:
                    break

            # Sort by date and time
            reminders.sort(key=lambda r: (r["date"], r["time"]))

            return {
                "ok": True,
                "reminders": reminders,
                "count": len(reminders),
                "total_available": len(lines),
            }

        except Exception as e:
            logger.error(f"List reminders failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def dedupe_reminders(self, preview: bool = False) -> Dict[str, Any]:
        """
        Remove duplicate reminder entries.

        Args:
            preview: If True, don't actually modify the file

        Returns:
            Result dict with deduplication stats
        """
        filepath = self._get_reminders_file()

        if not filepath.exists():
            return {
                "ok": True,
                "message": "No reminders file found.",
                "removed": 0,
            }

        try:
            with open(filepath, 'r', encoding='utf-8') as f:
                raw_lines = [line.rstrip("\n") for line in f.readlines()]
        except Exception as e:
            return {"ok": False, "error": f"Could not read reminders: {e}"}

        seen = set()
        kept = []
        removed = []

        for raw in raw_lines:
            line = str(raw or "").strip()
            if not line:
                continue

            # Normalize for comparison
            key = re.sub(r"\s+", " ", line).strip()

            if key in seen:
                removed.append(key)
                continue

            seen.add(key)
            kept.append(key)

        if not removed:
            return {
                "ok": True,
                "message": "No duplicate reminders found.",
                "removed": 0,
            }

        if preview:
            return {
                "ok": True,
                "preview": True,
                "removed": len(removed),
                "sample": removed[:8],
                "message": f"Would remove {len(removed)} duplicate rows.",
            }

        # Write deduped file
        try:
            with open(filepath, 'w', encoding='utf-8') as f:
                if kept:
                    f.write("\n".join(kept) + "\n")
                else:
                    f.write("")

            logger.info(f"Reminder dedupe complete: removed {len(removed)} duplicates")

            return {
                "ok": True,
                "message": f"Reminder dedupe complete. Removed {len(removed)} duplicate row(s).",
                "removed": len(removed),
            }

        except Exception as e:
            logger.error(f"Dedupe write failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def get_todays_reminders(self) -> List[Dict[str, Any]]:
        """Get all reminders for today."""
        result = self.list_reminders(include_past=False, limit=100)

        if not result.get("ok"):
            return []

        today = datetime.now().strftime("%Y-%m-%d")
        return [r for r in result.get("reminders", []) if r.get("date") == today]

    def get_reminder_stats(self) -> Dict[str, Any]:
        """Get reminder statistics."""
        filepath = self._get_reminders_file()

        if not filepath.exists():
            return {
                "total": 0,
                "today": 0,
                "upcoming": 0,
                "overdue": 0,
            }

        today = datetime.now().strftime("%Y-%m-%d")
        total = 0
        today_count = 0
        upcoming = 0
        overdue = 0

        try:
            with open(filepath, 'r', encoding='utf-8') as f:
                for line in f:
                    reminder = Reminder.from_line(line)
                    if reminder is None:
                        continue

                    total += 1

                    if reminder.date == today:
                        today_count += 1
                    elif reminder.date > today:
                        upcoming += 1
                    else:
                        overdue += 1

            return {
                "total": total,
                "today": today_count,
                "upcoming": upcoming,
                "overdue": overdue,
            }

        except Exception as e:
            logger.error(f"Reminder stats failed: {e}", exc_info=True)
            return {"error": str(e)}
```

```python
"""
Sage Notes Manager
Handles notes creation, listing, and tag management.
"""

import logging
import re
from datetime import datetime
from pathlib import Path
from typing import Dict, List, Optional, Any
from dataclasses import dataclass, field

from ..core.state import state
from ..core.config import get_config

logger = logging.getLogger(__name__)


@dataclass
class Note:
    """Represents a note entry."""
    timestamp: str
    text: str
    tags: List[str] = field(default_factory=list)
    workspace: str = "main"

    @classmethod
    def from_line(cls, line: str, workspace: str = "main") -> Optional['Note']:
        """Parse from file line format."""
        line = line.strip()
        if not line:
            return None

        # Extract timestamp [YYYY-MM-DD HH:MM]
        ts_match = re.match(r'^\[(\d{4}-\d{2}-\d{2}\s+\d{2}:\d{2})\]\s*(.*)$', line)
        if not ts_match:
            return None

        timestamp = ts_match.group(1)
        text = ts_match.group(2).strip()

        # Extract tags
        tags = re.findall(r'\[([^\]]+)\]', text)

        return cls(
            timestamp=timestamp,
            text=text,
            tags=tags,
            workspace=workspace,
        )


class NotesManager:
    """
    Manages notes with support for tags and workspaces.
    """

    # Allowed tag categories
    ALLOWED_TAGS = {
        "urgent", "home", "beba", "bajka", "health", "admin", "project", "test",
        "finance", "work", "personal", "idea", "reference",
    }

    def __init__(self, workspace_root: Optional[Path] = None):
        self.workspace_root = workspace_root or Path.cwd()
        self.config = get_config()
        self._jots_file: Optional[Path] = None
        self._projects_file: Optional[Path] = None

    def _get_jots_file(self, create: bool = False) -> Path:
        """Get the jots/notes file path."""
        if self._jots_file is None:
            self._jots_file = self.workspace_root / "NOTES" / "jots.txt"

        if create and not self._jots_file.parent.exists():
            self._jots_file.parent.mkdir(parents=True, exist_ok=True)

        return self._jots_file

    def _get_projects_file(self, create: bool = False) -> Path:
        """Get the projects notes file path."""
        if self._projects_file is None:
            self._projects_file = self.workspace_root / "NOTES" / "PROJECTS.txt"

        if create and not self._projects_file.parent.exists():
            self._projects_file.parent.mkdir(parents=True, exist_ok=True)

        return self._projects_file

    def _infer_tags(self, text: str) -> List[str]:
        """Infer tags from note text."""
        tags = []
        lowered = text.lower()

        inference_rules = [
            ("urgent", ["urgent", "asap", "critical", "immediately"]),
            ("health", ["health", "clinic", "doctor", "medical", "appointment", "nursing", "hospital"]),
            ("beba", ["beba"]),
            ("bajka", ["bajka"]),
            ("admin", ["admin", "system", "policy", "runtime", "config", "backup"]),
            ("home", ["home", "garage", "kitchen", "house"]),
            ("project", ["project", "build", "milestone", "sprint"]),
            ("test", ["test", "testing", "qa", "trial", "sandbox"]),
        ]

        for tag, keywords in inference_rules:
            if any(re.search(rf"\b{re.escape(keyword)}\b", lowered) for keyword in keywords):
                if tag not in tags:
                    tags.append(tag)

        return tags

    def add_note(
        self,
        text: str,
        tags: Optional[List[str]] = None,
    ) -> Dict[str, Any]:
        """
        Add a new note.

        Args:
            text: Note text
            tags: Optional list of tags

        Returns:
            Result dict with success status
        """
        # Parse explicit tags from text
        explicit_tags = []
        note_text = text

        # Check for [tag] prefix
        tag_match = re.match(r'^\[([^\[\]]+)\]\s*(.*)$', text)
        if tag_match:
            explicit_tags = [tag_match.group(1).strip().lower()]
            note_text = tag_match.group(2).strip()

        # Check for "tag <category>" prefix
        tag_prefix_match = re.match(r'^\s*(?:tag|tagged)\s*:?\s*([a-zA-Z]+)\b\s*(.*)$', text, re.IGNORECASE)
        if tag_prefix_match:
            candidate = tag_prefix_match.group(1).strip().lower()
            if candidate in self.ALLOWED_TAGS:
                explicit_tags = [candidate]
                note_text = tag_prefix_match.group(2).strip()

        # Use explicit tags or infer
        final_tags = explicit_tags if explicit_tags else (tags or [])

        if not final_tags:
            final_tags = self._infer_tags(note_text)

        # Filter to allowed tags
        final_tags = [t for t in final_tags if t.lower() in self.ALLOWED_TAGS or len(t) <= 20]

        # Build note line
        timestamp = datetime.now().strftime("%Y-%m-%d %H:%M")

        if final_tags:
            tags_str = " ".join(f"[{t}]" for t in final_tags)
            formatted_text = f"[{tags_str}] {note_text}".strip()
        else:
            formatted_text = note_text

        content = f"[{timestamp}] {formatted_text}\n"

        # Route project-tagged notes to projects file
        is_project = "project" in [t.lower() for t in final_tags]
        filepath = self._get_projects_file(create=True) if is_project else self._get_jots_file(create=True)

        try:
            with open(filepath, 'a', encoding='utf-8') as f:
                f.write(content)

            logger.info(f"Note added to {filepath.name}: {formatted_text[:60]}...")

            return {
                "ok": True,
                "message": f"Note saved to {filepath.name}",
                "path": str(filepath.relative_to(self.workspace_root)),
                "tags": final_tags,
                "timestamp": timestamp,
            }

        except Exception as e:
            logger.error(f"Add note failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def list_notes(
        self,
        tag_filter: Optional[str] = None,
        limit: int = 50,
        include_projects: bool = True,
    ) -> Dict[str, Any]:
        """
        List notes with optional tag filter.

        Args:
            tag_filter: Filter by tag
            limit: Maximum notes to return
            include_projects: If True, include project notes

        Returns:

        """
        notes: List[Dict[str, Any]] = []
        source_files = [self._get_jots_file()]

        if include_projects:
            source_files.append(self._get_projects_file())

        for filepath in source_files:

                continue

            try:
                with open(filepath, 'r', encoding='utf-8') as f:
                    lines = f.readlines()

                for line in lines:
                    note = Note.from_line(line)

                    if note is None:
                        continue

                    # Filter by tag
                    if tag_filter:
                        tag_lower = tag_filter.lower()
                        if not any(tag_lower in t.lower() for t in note.tags):
                            continue

                    notes.append({
                        "timestamp": note.timestamp,
                        "text": note.text,
                        "tags": note.tags,
                        "workspace": note.workspace,
                        "source": filepath.name,
                    })

                    if len(notes) >= limit:
                        break

            except Exception as e:
                logger.error(f"Read notes failed ({filepath}): {e}")
                continue

        # Sort by timestamp (newest first)
        notes.sort(key=lambda n: n["timestamp"], reverse=True)

        return {
            "ok": True,
            "notes": notes,
            "count": len(notes),
            "tag_filter": tag_filter,
        }

    def get_tags_summary(self, tag_filter: Optional[str] = None) -> Dict[str, Any]:

        projects_file = self._get_projects_file()

        tagged_lines: List[str] = []

                for line in lines:
                    note = Note.from_line(line)

                    if note is None or not note.tags:
                        continue

                    # Count tags
                    for tag in note.tags:
                        tag_lower = tag.lower()
                        tag_counts[tag_lower] = tag_counts.get(tag_lower, 0) + 1

                    # Store tagged line
                    tagged_lines.append(f"[{filepath.name}] {note.text}")

            except Exception as e:

                continue

        # Sort by count
        sorted_tags = sorted(tag_counts.items(), key=lambda x: (-x[1], x[0]))

        # If filtering by tag, return matching entries
        if tag_filter:
            tag_lower = tag_filter.lower()
            matches = [line for line in tagged_lines if tag_lower in line.lower()]

            return {
                "ok": True,
                "tag": tag_filter,
                "count": tag_counts.get(tag_lower, 0),
                "entries": matches[:50],
                "total_entries": len(matches),
            }

        return {
            "ok": True,
            "tags": [{"tag": tag, "count": count} for tag, count in sorted_tags],
            "total_tags": len(tag_counts),
            "recent_tagged": tagged_lines[-20:],
        }
```

```python
# -*- coding: utf-8 -*-
"""
Sage Search Engine
Handles file and content search with synonym expansion and fuzzy matching.
"""

import logging
import re
import difflib
from pathlib import Path
from typing import Dict, List, Optional, Any, Set

from ..core.state import state
from ..core.config import get_config

logger = logging.getLogger(__name__)


class SearchEngine:
    """
    Provides smart context-aware search with synonyms and fuzzy matching.
    """

    # Synonym expansion for common terms
    SYNONYMS = {
        'phone': ['phone', 'tel', 'contact', 'number', 'call'],
        'email': ['email', 'e-mail', 'contact', 'mail'],
        'address': ['address', 'location', 'addr'],
        'number': ['number', 'no', 'num', '#'],
        'id': ['id', 'identification', 'number'],
        'date': ['date', 'when', 'scheduled', 'appointment'],
        'time': ['time', 'when', 'at', 'clock'],
    }

    def __init__(self, workspace_root: Optional[Path] = None):
        self.workspace_root = workspace_root or Path.cwd()
        self.config = get_config()

    def search_file(
        self,
        keyword: str,
        filepath: str,
        case_insensitive: bool = True,
        limit: int = 10,
    ) -> Dict[str, Any]:
        """
        Search for keyword in a specific file.

        Args:
            keyword: Search keyword
            filepath: Path to file
            case_insensitive: If True, ignore case
            limit: Maximum matches to return

        Returns:
            Dict with matches list and metadata
        """
        try:
            target = self.workspace_root / filepath

            if not target.exists():
                return {"ok": False, "error": f"File not found: {filepath}"}

            if not target.is_file():
                return {"ok": False, "error": f"Not a file: {filepath}"}

            with open(target, 'r', encoding='utf-8', errors='replace') as f:
                lines = f.readlines()

            matches: List[Dict[str, Any]] = []
            search_keyword = keyword.lower() if case_insensitive else keyword

            for i, line in enumerate(lines, 1):
                search_line = line.lower() if case_insensitive else line

                if search_keyword in search_line:
                    matches.append({
                        "line_number": i,
                        "content": line.strip(),
                    })

                    if len(matches) >= limit:
                        break

            if not matches:
                return {
                    "ok": True,
                    "matches": [],
                    "count": 0,
                    "message": f"No matches for '{keyword}' in {filepath}",
                }

            return {
                "ok": True,
                "matches": matches,
                "count": len(matches),
                "file": filepath,
                "keyword": keyword,
            }

        except Exception as e:
            logger.error(f"Search file failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def search_directory(
        self,
        keyword: str,
        directory: str = ".",
        file_pattern: str = "*.txt",
        limit: int = 15,
    ) -> Dict[str, Any]:
        """
        Search for keyword in all matching files in a directory.

        Args:
            keyword: Search keyword
            directory: Directory to search
            file_pattern: Glob pattern for files
            limit: Maximum total matches

        Returns:
            Dict with matches list and metadata
        """
        try:
            target_dir = self.workspace_root / directory

            if not target_dir.exists():
                return {"ok": False, "error": f"Directory not found: {directory}"}

            if not target_dir.is_dir():
                return {"ok": False, "error": f"Not a directory: {directory}"}

            all_matches: List[Dict[str, Any]] = []
            search_keyword = keyword.lower()

            # Expand keywords with synonyms
            expanded_keywords = self._expand_keywords(keyword)

            for filepath in target_dir.glob(file_pattern):
                if not filepath.is_file():
                    continue

                try:
                    with open(filepath, 'r', encoding='utf-8', errors='replace') as f:
                        lines = f.readlines()

                    for i, line in enumerate(lines, 1):
                        line_lower = line.lower()

                        # Check direct match
                        if search_keyword in line_lower:
                            all_matches.append({
                                "file": str(filepath.relative_to(self.workspace_root)),
                                "line_number": i,
                                "content": line.strip(),
                                "match_type": "direct",
                            })

                            if len(all_matches) >= limit:
                                break

                        # Check synonym matches
                        else:
                            for syn in expanded_keywords:
                                if syn != search_keyword and syn in line_lower:
                                    all_matches.append({
                                        "file": str(filepath.relative_to(self.workspace_root)),
                                        "line_number": i,
                                        "content": line.strip(),
                                        "match_type": f"synonym:{syn}",
                                    })

                                    if len(all_matches) >= limit:
                                        break

                    if len(all_matches) >= limit:
                        break

                except Exception as e:
                    logger.debug(f"Search file skipped ({filepath}): {e}")
                    continue

            if not all_matches:
                return {
                    "ok": True,
                    "matches": [],
                    "count": 0,
                    "message": f"No matches for '{keyword}' in {directory}",
                }

            return {
                "ok": True,
                "matches": all_matches,
                "count": len(all_matches),
                "directory": directory,
                "keyword": keyword,
                "truncated": len(all_matches) >= limit,
            }

        except Exception as e:
            logger.error(f"Search directory failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def search_smart(
        self,
        keyword: str,
        search_path: Optional[str] = None,
        use_fuzzy: bool = True,
        limit: int = 15,
    ) -> Dict[str, Any]:
        """
        Smart context-aware search with multiple strategies.

        Args:
            keyword: Search keyword(s)
            search_path: Optional path (file or directory)
            use_fuzzy: Enable fuzzy matching as fallback
            limit: Maximum matches to return

        Returns:
            Dict with matches list and metadata
        """
        # Default search path
        if not search_path:
            search_path = "HOME/ts-keep.txt"

        target = self.workspace_root / search_path

        # If directory, search all .txt files
        if target.exists() and target.is_dir():
            return self.search_directory(keyword, search_path, "*.txt", limit)

        # Single file search
        if not target.exists():
            return {"ok": False, "error": f"File not found: {search_path}"}

        try:
            with open(target, 'r', encoding='utf-8', errors='replace') as f:
                lines = f.readlines()

            matches: List[Dict[str, Any]] = []
            matched_indices: Set[int] = set()

            search_words = keyword.lower().split()
            expanded_words = self._expand_keywords(keyword)

            # Pass 1: Direct match
            for i, line in enumerate(lines):
                line_lower = line.lower()

                if keyword.lower() in line_lower:
                    matched_indices.add(i)
                    matches.append({
                        "file": search_path,
                        "line_number": i + 1,
                        "content": line.strip(),
                        "match_type": "direct",
                    })

            # Pass 2: Multi-word proximity search
            if len(search_words) > 1:
                for i, line in enumerate(lines):
                    if i in matched_indices:
                        continue

                    line_lower = line.lower()
                    found_words = [w for w in search_words if w in line_lower]

                    if found_words:
                        # Look for other words in nearby lines
                        context_range = range(max(0, i-3), min(len(lines), i+4))
                        nearby_text = " ".join(lines[j].lower() for j in context_range)

                        if all(w in nearby_text for w in search_words):
                            matched_indices.add(i)
                            matches.append({
                                "file": search_path,
                                "line_number": i + 1,
                                "content": line.strip(),
                                "match_type": "proximity",
                            })

            # Pass 3: Synonym-enhanced search
            if not matches:
                for i, line in enumerate(lines):
                    if i in matched_indices:
                        continue

                    line_lower = line.lower()

                    if any(exp_word in line_lower for exp_word in expanded_words):
                        # Check if a key term is nearby
                        proper_nouns = [w for w in search_words if w[0].isupper() or len(w) > 3]

                        if proper_nouns:
                            context_range = range(max(0, i-2), min(len(lines), i+3))
                            nearby_text = " ".join(lines[j].lower() for j in context_range)

                            if any(noun.lower() in nearby_text for noun in proper_nouns):
                                matched_indices.add(i)
                                matches.append({
                                    "file": search_path,
                                    "line_number": i + 1,
                                    "content": line.strip(),
                                    "match_type": "synonym",
                                })

            # Pass 4: Fuzzy matching (last resort)
            if not matches and use_fuzzy:
                for i, line in enumerate(lines):
                    if i in matched_indices:
                        continue

                    words = [w.lower().strip('.,!?:;()[]"\'') for w in line.split()]

                    for search_word in search_words:
                        if len(search_word) < 4:
                            continue

                        close = difflib.get_close_matches(search_word, words, n=1, cutoff=0.74)

                        if close:
                            matches.append({
                                "file": search_path,
                                "line_number": i + 1,
                                "content": line.strip(),
                                "match_type": f"fuzzy:{close[0]}",
                            })
                            matched_indices.add(i)
                            break

            if not matches:
                return {
                    "ok": True,
                    "matches": [],
                    "count": 0,
                    "message": f"No matches for '{keyword}' in {search_path}",
                    "tip": "Try person's name only, or key terms like 'phone', 'email', 'address'",
                }

            return {
                "ok": True,
                "matches": matches[:limit],
                "count": len(matches),
                "file": search_path,
                "keyword": keyword,
                "truncated": len(matches) > limit,
            }

        except Exception as e:
            logger.error(f"Smart search failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def _expand_keywords(self, keyword: str) -> List[str]:
        """Expand search keywords with synonyms."""
        normalized = re.sub(r"[^a-z0-9\s]", " ", keyword.lower())
        normalized = re.sub(r"\s+", " ", normalized).strip()

        search_words = [w for w in normalized.split() if w]

        if not search_words:
            return []

        expanded = []

        for word in search_words:
            expanded.append(word)

            # Add synonyms
            for key, syns in self.SYNONYMS.items():
                if word in syns:
                    expanded.extend([s for s in syns if s != word])
                    break

            # Add key itself
            for key, syns in self.SYNONYMS.items():
                if word == key:
                    expanded.extend(syns)
                    break

        return list(set(expanded))


# Global search engine instance
search_engine = SearchEngine()


def get_search_engine() -> SearchEngine:
    """Get the global search engine instance."""
    return search_engine
```

# -*- coding: utf-8 -*-
"""
Sage Memory Manager
Handles long-term memory storage and retrieval.
"""

import logging
import re
from datetime import datetime
from pathlib import Path
from typing import Dict, List, Optional, Any, Set

from ..core.state import state
from ..core.config import get_config

logger = logging.getLogger(__name__)


class MemoryManager:
    """
    Manages long-term memory for the AI assistant.
    Provides add, list, search, dedupe, and remove operations.
    """

    def __init__(self, workspace_root: Optional[Path] = None):
        self.workspace_root = workspace_root or Path.cwd()
        self.config = get_config()
        self._memory_file: Optional[Path] = None

    def _get_memory_file(self, create: bool = False) -> Path:
        """Get the memory file path."""
        if self._memory_file is None:
            self._memory_file = self.workspace_root / "NOTES" / "model-memory.txt"

        if create and not self._memory_file.parent.exists():
            self._memory_file.parent.mkdir(parents=True, exist_ok=True)

        return self._memory_file

    def _strip_timestamp(self, line: str) -> str:
        """Remove timestamp from memory line."""
        return re.sub(r'^\[\d{4}-\d{2}-\d{2}\s+\d{2}:\d{2}\]\s*', '', line).strip()

    def _normalize_text(self, value: str) -> str:
        """Normalize text for comparison."""
        return re.sub(r'\s+', ' ', value).strip().lower()

    def add_memory(self, text: str) -> Dict[str, Any]:
        """
        Add a new memory entry.

        Args:
            text: Memory text

        Returns:
            Result dict with success status
        """
        if not text or not text.strip():
            return {"ok": False, "error": "Memory text cannot be empty."}

        # Check for duplicates
        normalized_new = self._normalize_text(text)

        memory_file = self._get_memory_file()

        if memory_file.exists():
            try:
                with open(memory_file, 'r', encoding='utf-8') as f:
                    for line in f:
                        existing = self._normalize_text(self._strip_timestamp(line))
                        if existing and existing == normalized_new:
                            return {
                                "ok": True,
                                "skipped": True,
                                "message": "Duplicate memory skipped.",
                            }
            except Exception as e:
                logger.error(f"Check duplicates failed: {e}")

        # Add memory
        timestamp = datetime.now().strftime("%Y-%m-%d %H:%M")
        entry = f"[{timestamp}] {text}\n"

        try:
            with open(memory_file, 'a', encoding='utf-8') as f:
                f.write(entry)

            logger.info(f"Memory added: {text[:60]}...")

            return {
                "ok": True,
                "message": "Memory saved.",
                "timestamp": timestamp,
            }

        except Exception as e:
            logger.error(f"Add memory failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def list_memories(self, limit: int = 50) -> Dict[str, Any]:
        """
        List recent memory entries.

        Args:
            limit: Maximum memories to return

        Returns:
            Dict with memories list and metadata
        """
        memory_file = self._get_memory_file()

        if not memory_file.exists():
            return {
                "ok": True,
                "memories": [],
                "count": 0,
                "message": "No memories yet. Use: mem-add [text]",
            }

        try:
            with open(memory_file, 'r', encoding='utf-8') as f:
                lines = [line.rstrip("\n") for line in f if line.strip()]

            if not lines:
                return {
                    "ok": True,
                    "memories": [],
                    "count": 0,
                    "message": "No memories yet.",
                }

            # Get most recent
            recent = lines[-limit:] if len(lines) > limit else lines

            memories = []
            for line in recent:
                memory = self._parse_memory_line(line)
                if memory:
                    memories.append(memory)

            return {
                "ok": True,
                "memories": memories,
                "count": len(memories),
                "total": len(lines),
            }

        except Exception as e:
            logger.error(f"List memories failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def search_memories(self, query: str, limit: int = 30) -> Dict[str, Any]:
        """
        Search memories by keyword.

        Args:
            query: Search query
            limit: Maximum matches to return

        Returns:
            Dict with matching memories
        """
        if not query or not query.strip():
            return {"ok": False, "error": "Search query cannot be empty."}

        memory_file = self._get_memory_file()

        if not memory_file.exists():
            return {
                "ok": True,
                "memories": [],
                "count": 0,
                "message": "No memories yet.",
            }

        query_lower = query.lower()

        try:
            with open(memory_file, 'r', encoding='utf-8') as f:
                lines = [line.rstrip("\n") for line in f if line.strip()]

            matches = []
            for line in lines:
                if query_lower in line.lower():
                    memory = self._parse_memory_line(line)
                    if memory:
                        matches.append(memory)

                        if len(matches) >= limit:
                            break

            if not matches:
                return {
                    "ok": True,
                    "memories": [],
                    "count": 0,
                    "message": f"No memory matches for: {query}",
                }

            return {
                "ok": True,
                "memories": matches,
                "count": len(matches),
                "query": query,
            }

        except Exception as e:
            logger.error(f"Search memories failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def dedupe_memories(self) -> Dict[str, Any]:
        """
        Remove duplicate memory entries (timestamp-agnostic).

        Returns:
            Result dict with deduplication stats
        """
        memory_file = self._get_memory_file()

        if not memory_file.exists():
            return {
                "ok": True,
                "message": "No memory file found.",
                "removed": 0,
            }

        try:
            with open(memory_file, 'r', encoding='utf-8') as f:
                lines = [line.rstrip("\n") for line in f if line.strip()]

            if not lines:
                return {
                    "ok": True,
                    "message": "No memories to dedupe.",
                    "removed": 0,
                }

            seen: Set[str] = set()
            deduped = []
            removed = 0

            for line in lines:
                payload = self._strip_timestamp(line)
                normalized = self._normalize_text(payload)

                if not normalized:
                    continue

                if normalized in seen:
                    removed += 1
                    continue

                seen.add(normalized)
                deduped.append(line)

            if removed == 0:
                return {
                    "ok": True,
                    "message": "No duplicate memories found.",
                    "removed": 0,
                }

            # Write deduped file
            with open(memory_file, 'w', encoding='utf-8') as f:
                f.write("\n".join(deduped) + ("\n" if deduped else ""))

            logger.info(f"Memory dedupe complete: removed {removed} duplicates")

            return {
                "ok": True,
                "message": f"Memory dedupe complete. Removed {removed} duplicate entr{'y' if removed == 1 else 'ies'}.",
                "removed": removed,
            }

        except Exception as e:
            logger.error(f"Dedupe memories failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def remove_memories(self, query: str) -> Dict[str, Any]:
        """
        Remove memories matching a keyword.

        Args:
            query: Keyword to match for removal

        Returns:
            Result dict with removal stats
        """
        if not query or not query.strip():
            return {"ok": False, "error": "Query cannot be empty."}

        memory_file = self._get_memory_file()

        if not memory_file.exists():
            return {
                "ok": True,
                "message": "No memory file found.",
                "removed": 0,
            }

        query_lower = query.lower()

        try:
            with open(memory_file, 'r', encoding='utf-8') as f:
                lines = [line.rstrip("\n") for line in f if line.strip()]

            if not lines:
                return {
                    "ok": True,
                    "message": "No memories to remove.",
                    "removed": 0,
                }

            kept = []
            removed_count = 0

            for line in lines:
                payload = self._strip_timestamp(line).lower()

                if query_lower in payload:
                    removed_count += 1
                else:
                    kept.append(line)

            if removed_count == 0:
                return {
                    "ok": True,
                    "message": f"No memory matches for removal: {query}",
                    "removed": 0,
                }

            # Write updated file
            with open(memory_file, 'w', encoding='utf-8') as f:
                f.write("\n".join(kept) + ("\n" if kept else ""))

            logger.info(f"Removed {removed_count} memories matching: {query}")

            return {
                "ok": True,
                "message": f"Removed {removed_count} memory entr{'y' if removed_count == 1 else 'ies'} matching '{query}'.",
                "removed": removed_count,
            }

        except Exception as e:
            logger.error(f"Remove memories failed: {e}", exc_info=True)
            return {"ok": False, "error": str(e)}

    def retrieve_relevant(self, user_text: str, max_lines: int = 5, max_chars: int = 700) -> str:
        """
        Retrieve memories relevant to user text.

        Args:
            user_text: User's input text
            max_lines: Maximum memories to return
            max_chars: Maximum total characters

        Returns:
            Concatenated relevant memories or empty string
        """
        memory_file = self._get_memory_file()

        if not memory_file.exists():
            return ""

        # Extract keywords from user text
        words = set(w for w in re.findall(r"\b\w+\b", user_text.lower()) if len(w) > 3)

        if not words:
            return ""

        matches = []
        total_chars = 0

        try:
            with open(memory_file, 'r', encoding='utf-8') as f:
                for line in f:
                    low = line.lower().strip()

                    if not low:
                        continue

                    # Skip system lines
                    if any(token in low for token in ("assistant:", "user:", "system:", "cmd:", "```")):
                        continue

                    # Check for keyword match
                    if any(w in low for w in words):
                        candidate = line.strip()

                        if total_chars + len(candidate) > max_chars:
                            break

                        matches.append(candidate)
                        total_chars += len(candidate)

                        if len(matches) >= max_lines:
                            break

            return "\n".join(matches) if matches else ""

        except Exception as e:
            logger.error(f"Retrieve relevant failed: {e}")
            return ""

    def _parse_memory_line(self, line: str) -> Optional[Dict[str, Any]]:
        """Parse a memory line into structured data."""
        line = line.strip()
        if not line:
            return None

        ts_match = re.match(r'^\[(\d{4}-\d{2}-\d{2}\s+\d{2}:\d{2})\]\s*(.*)$', line)

        if not ts_match:
            return None

        return {
            "timestamp": ts_match.group(1),
            "text": ts_match.group(2).strip(),
            "raw": line,
        }


# Global memory manager instance
memory_manager = MemoryManager()


def get_memory_manager() -> MemoryManager:
    """Get the global memory manager instance."""
    return memory_manager

```python
# -*- coding: utf-8 -*-
"""
Sage Model Client
Handles direct communication with LLM backends (Ollama, Foundry, Cloud).
"""

import json
import logging
import time
import requests
from typing import Dict, List, Optional, Any, Callable, Generator
from dataclasses import dataclass
from enum import Enum

from ..core.state import state
from ..core.config import get_config, ModelProfile

logger = logging.getLogger(__name__)


class BackendType(Enum):
    """Supported LLM backend types."""
    OLLAMA = "ollama"
    FOUNDRY = "foundry"
    CLOUD = "cloud"


@dataclass
class ModelResponse:
    """Response from a model call."""
    text: str
    model: str
    backend: str
    duration_ms: int
    tokens_predicted: int = 0
    error: Optional[str] = None
    warning: Optional[str] = None


@dataclass
class StreamChunk:
    """A single chunk from a streaming response."""
    text: str
    done: bool = False


class ModelClient:
    """
    Unified client for calling LLM models across different backends.
    Supports both streaming and non-streaming modes.
    """

    def __init__(self):
        self.config = get_config()
        self._session = requests.Session()
        self._session.headers.update({
            "Content-Type": "application/json",
            "User-Agent": "Sage-Lite/1.0",
        })

    def call(
        self,
        profile: ModelProfile,
        prompt: str,
        system_prompt: Optional[str] = None,
        messages: Optional[List[Dict[str, str]]] = None,
        stream: bool = False,
        stream_callback: Optional[Callable[[str], None]] = None,
        extra_context: Optional[str] = None,
        timeout_override: Optional[int] = None,
    ) -> ModelResponse:
        """
        Call a model with the given profile.

        Args:
            profile: Model profile with backend, endpoint, model name, etc.
            prompt: User prompt text
            system_prompt: Optional system instruction
            messages: Optional chat messages list (for chat API)
            stream: Enable streaming response
            stream_callback: Callback for stream chunks
            extra_context: Additional context to inject
            timeout_override: Override timeout from profile

        Returns:
            ModelResponse with generated text and metadata
        """
        backend = profile.backend.lower()

        if backend == BackendType.OLLAMA.value:
            return self._call_ollama(
                profile=profile,
                prompt=prompt,
                system_prompt=system_prompt,
                messages=messages,
                stream=stream,
                stream_callback=stream_callback,
                extra_context=extra_context,
                timeout_override=timeout_override,
            )
        elif backend == BackendType.FOUNDRY.value:
            return self._call_foundry(
                profile=profile,
                prompt=prompt,
                system_prompt=system_prompt,
                messages=messages,
                stream=stream,
                stream_callback=stream_callback,
                extra_context=extra_context,
                timeout_override=timeout_override,
            )
        elif backend == BackendType.CLOUD.value:
            return self._call_cloud(
                profile=profile,
                prompt=prompt,
                system_prompt=system_prompt,
                messages=messages,
                stream=stream,
                stream_callback=stream_callback,
                extra_context=extra_context,
                timeout_override=timeout_override,
            )
        else:
            return ModelResponse(
                text="",
                model=profile.name,
                backend=backend,
                duration_ms=0,
                error=f"Unsupported backend: {backend}",
            )

    def _call_ollama(
        self,
        profile: ModelProfile,
        prompt: str,
        system_prompt: Optional[str] = None,
        messages: Optional[List[Dict[str, str]]] = None,
        stream: bool = False,
        stream_callback: Optional[Callable[[str], None]] = None,
        extra_context: Optional[str] = None,
        timeout_override: Optional[int] = None,
    ) -> ModelResponse:
        """Call Ollama API (generate or chat)."""
        start_time = time.perf_counter()

        endpoint = profile.endpoint.rstrip("/")
        if not endpoint.endswith("/api/generate") and not endpoint.endswith("/api/chat"):
            # Default to chat API for multi-turn, generate for single prompt
            if messages:
                endpoint += "/api/chat"
            else:
                endpoint += "/api/generate"

        timeout = timeout_override or profile.timeout_seconds

        # Build payload
        if "/api/chat" in endpoint or messages:
            payload = self._build_chat_payload(
                profile=profile,
                messages=messages,
                prompt=prompt,
                system_prompt=system_prompt,
                extra_context=extra_context,
                stream=stream,
            )
        else:
            payload = self._build_generate_payload(
                profile=profile,
                prompt=prompt,
                system_prompt=system_prompt,
                extra_context=extra_context,
                stream=stream,
            )

        try:
            logger.debug(f"Ollama request: {endpoint} (model={profile.name}, stream={stream})")

            if stream:
                text, tokens = self._stream_ollama(endpoint, payload, stream_callback, timeout)
            else:
                text, tokens = self._request_ollama(endpoint, payload, timeout)

            duration_ms = int((time.perf_counter() - start_time) * 1000)

            logger.debug(f"Ollama response: {len(text)} chars in {duration_ms}ms")

            return ModelResponse(
                text=text.strip() if text else "",
                model=profile.name,
                backend="ollama",
                duration_ms=duration_ms,
                tokens_predicted=tokens,
            )

        except requests.exceptions.Timeout:
            duration_ms = int((time.perf_counter() - start_time) * 1000)
            logger.warning(f"Ollama timeout after {timeout}s")
            return ModelResponse(
                text="",
                model=profile.name,
                backend="ollama",
                duration_ms=duration_ms,
                warning=f"Model timed out after {timeout}s",
            )
        except requests.exceptions.RequestException as e:
            duration_ms = int((time.perf_counter() - start_time) * 1000)
            logger.error(f"Ollama request failed: {e}")
            return ModelResponse(
                text="",
                model=profile.name,
                backend="ollama",
                duration_ms=duration_ms,
                error=str(e),
            )
        except Exception as e:
            duration_ms = int((time.perf_counter() - start_time) * 1000)
            logger.error(f"Ollama unexpected error: {e}", exc_info=True)
            return ModelResponse(
                text="",
                model=profile.name,
                backend="ollama",
                duration_ms=duration_ms,
                error=str(e),
            )

    def _build_generate_payload(
        self,
        profile: ModelProfile,
        prompt: str,
        system_prompt: Optional[str],
        extra_context: Optional[str],
        stream: bool,
    ) -> Dict[str, Any]:
        """Build Ollama generate API payload."""
        # Combine system prompt and context
        full_system = ""
        if system_prompt:
            full_system += system_prompt.strip() + "\n\n"
        if extra_context:
            full_system += f"[CONTEXT]\n{extra_context}\n\n"

        full_prompt = prompt
        if full_system:
            full_prompt = f"{full_system}{prompt}"

        return {
            "model": profile.name,
            "prompt": full_prompt,
            "system": system_prompt.strip() if system_prompt and not extra_context else "",
            "stream": stream,
            "keep_alive": self.config.runtime.ollama_keep_alive if hasattr(self.config, 'runtime') else "90s",
            "options": {
                "temperature": profile.temperature,
                "top_p": 0.95,
                "frequency_penalty": 0.1,
                "num_predict": profile.num_predict,
            },
        }

    def _build_chat_payload(
        self,
        profile: ModelProfile,
        messages: Optional[List[Dict[str, str]]],
        prompt: str,
        system_prompt: Optional[str],
        extra_context: Optional[str],
        stream: bool,
    ) -> Dict[str, Any]:
        """Build Ollama chat API payload."""
        chat_messages = []

        # Add system message
        system_content = ""
        if system_prompt:
            system_content += system_prompt.strip()
        if extra_context:
            if system_content:
                system_content += "\n\n"
            system_content += f"[CONTEXT]\n{extra_context}"

        if system_content:
            chat_messages.append({"role": "system", "content": system_content})

        # Add existing messages or create from prompt
        if messages:
            chat_messages.extend(messages)
        else:
            chat_messages.append({"role": "user", "content": prompt})

        return {
            "model": profile.name,
            "messages": chat_messages,
            "stream": stream,
            "keep_alive": self.config.runtime.ollama_keep_alive if hasattr(self.config, 'runtime') else "90s",
            "options": {
                "temperature": profile.temperature,
                "top_p": 0.95,
                "frequency_penalty": 0.1,
                "num_predict": profile.num_predict,
            },
        }

    def _request_ollama(
        self,
        endpoint: str,
        payload: Dict[str, Any],
        timeout: int,
    ) -> tuple:
        """Make non-streaming Ollama request."""
        resp = self._session.post(endpoint, json=payload, timeout=timeout, stream=False)
        resp.raise_for_status()

        data = resp.json()

        # Extract text from response
        text = ""
        tokens = 0

        if "response" in data:
            text = data.get("response", "")
        elif "message" in data:
            text = data.get("message", {}).get("content", "")

        tokens = data.get("eval_count", 0)

        return text, tokens

    def _stream_ollama(
        self,
        endpoint: str,
        payload: Dict[str, Any],
        callback: Optional[Callable[[str], None]],
        timeout: int,
    ) -> tuple:
        """Make streaming Ollama request."""
        resp = self._session.post(endpoint, json=payload, timeout=timeout, stream=True)
        resp.raise_for_status()

        chunks = []
        tokens = 0

        for line in resp.iter_lines(chunk_size=1, decode_unicode=True):
            if not line:
                continue

            line = line.strip()

            # Handle SSE format
            if line.startswith("data:"):
                line = line[5:].strip()

            if line in {"[DONE]", "done"}:
                break

            try:
                event = json.loads(line)
            except json.JSONDecodeError:
                # Some providers emit plain text
                if line:
                    chunks.append(line)
                    if callback:
                        callback(line)
                continue

            # Extract text from event
            part = ""

            if "message" in event:
                part = event.get("message", {}).get("content", "")
            elif "response" in event:
                part = event.get("response", "")

            if part:
                chunks.append(part)
                if callback:
                    callback(part)

            if event.get("done", False):
                tokens = event.get("eval_count", 0)
                break

        return "".join(chunks), tokens

    def _call_foundry(
        self,
        profile: ModelProfile,
        prompt: str,
        system_prompt: Optional[str] = None,
        messages: Optional[List[Dict[str, str]]] = None,
        stream: bool = False,
        stream_callback: Optional[Callable[[str], None]] = None,
        extra_context: Optional[str] = None,
        timeout_override: Optional[int] = None,
    ) -> ModelResponse:
        """Call Foundry API (OpenAI-compatible)."""
        start_time = time.perf_counter()

        endpoint = profile.endpoint.rstrip("/")
        timeout = timeout_override or profile.timeout_seconds

        # Normalize endpoint
        if not endpoint.endswith("/chat/completions"):
            if endpoint.endswith("/v1"):
                endpoint = endpoint[:-3] + "/openai/chat/completions"
            else:
                endpoint += "/chat/completions"

        # Build messages
        chat_messages = []

        # System message
        system_content = ""
        if system_prompt:
            system_content += system_prompt.strip()
        if extra_context:
            if system_content:
                system_content += "\n\n"
            system_content += f"[CONTEXT]\n{extra_context}"

        if system_content:
            chat_messages.append({"role": "system", "content": system_content})

        # User messages
        if messages:
            chat_messages.extend(messages)
        else:
            chat_messages.append({"role": "user", "content": prompt})

        payload = {
            "model": profile.name,
            "messages": chat_messages,
            "temperature": profile.temperature,
            "top_p": 0.95,
            "frequency_penalty": 0.1,
            "max_tokens": profile.num_predict,
            "stream": stream,
        }

        try:
            logger.debug(f"Foundry request: {endpoint} (model={profile.name})")

            if stream:
                text, tokens = self._stream_foundry(endpoint, payload, stream_callback, timeout)
            else:
                text, tokens = self._request_foundry(endpoint, payload, timeout)

            duration_ms = int((time.perf_counter() - start_time) * 1000)

            return ModelResponse(
                text=text.strip() if text else "",
                model=profile.name,
                backend="foundry",
                duration_ms=duration_ms,
                tokens_predicted=tokens,
            )

        except requests.exceptions.Timeout:
            duration_ms = int((time.perf_counter() - start_time) * 1000)
            return ModelResponse(
                text="",
                model=profile.name,
                backend="foundry",
                duration_ms=duration_ms,
                warning=f"Model timed out after {timeout}s",
            )
        except Exception as e:
            duration_ms = int((time.perf_counter() - start_time) * 1000)
            logger.error(f"Foundry request failed: {e}", exc_info=True)
            return ModelResponse(
                text="",
                model=profile.name,
                backend="foundry",
                duration_ms=duration_ms,
                error=str(e),
            )

    def _request_foundry(
        self,
        endpoint: str,
        payload: Dict[str, Any],
        timeout: int,
    ) -> tuple:
        """Make non-streaming Foundry request."""
        resp = self._session.post(endpoint, json=payload, timeout=timeout, stream=False)
        resp.raise_for_status()

        data = resp.json()

        text = ""
        tokens = 0

        choices = data.get("choices", [])
        if choices:
            first = choices[0]
            message = first.get("message", {})
            text = message.get("content", "")
            tokens = first.get("usage", {}).get("completion_tokens", 0)

        return text, tokens

    def _stream_foundry(
        self,
        endpoint: str,
        payload: Dict[str, Any],
        callback: Optional[Callable[[str], None]],
        timeout: int,
    ) -> tuple:
        """Make streaming Foundry request."""
        payload["stream"] = True

        resp = self._session.post(endpoint, json=payload, timeout=timeout, stream=True)
        resp.raise_for_status()

        chunks = []
        tokens = 0

        for line in resp.iter_lines(chunk_size=1, decode_unicode=True):
            if not line:
                continue

            line = line.strip()

            if line.startswith("data:"):
                line = line[5:].strip()

            if line == "[DONE]" or line == "done":
                break

            try:
                event = json.loads(line)
            except json.JSONDecodeError:
                continue

            choices = event.get("choices", [])
            if choices:
                delta = choices[0].get("delta", {})
                part = delta.get("content", "")

                if part:
                    chunks.append(part)
                    if callback:
                        callback(part)

                if choices[0].get("finish_reason"):
                    usage = event.get("usage", {})
                    tokens = usage.get("completion_tokens", 0)
                    break

        return "".join(chunks), tokens

    def _call_cloud(

        **kwargs,
    ) -> ModelResponse:
        """
        Call cloud-based model (placeholder for future expansion).

        """
        # For now, treat cloud as Ollama with different endpoint
        return self._call_ollama(profile, prompt, **kwargs)

    def warmup_model(self, profile: ModelProfile) -> Dict[str, Any]:

                "error": "",
                "latency_ms": elapsed_ms,

        except Exception as e:

            return {
                "ok": False,
                "model": profile.name,
                "error": str(e),
                "latency_ms": elapsed_ms,
```

## 3. `agents/context.py` (Context Manager)

```python
# -*- coding: utf-8 -*-
"""
Sage Context Manager
Handles conversation history, memory injection, and context building.
"""

import logging
import re
from datetime import datetime
from pathlib import Path
from typing import Dict, List, Optional, Any, Tuple
from dataclasses import dataclass, field

from ..core.state import state
from ..core.config import get_config

logger = logging.getLogger(__name__)


@dataclass
class ContextEntry:
    """A single entry in the context stack."""
    role: str  # "user" or "sage"
    text: str
    timestamp: datetime = field(default_factory=datetime.now)
    summary: Optional[str] = None


@dataclass
class ContextStack:
    """Rolling context stack for conversation history."""
    entries: List[ContextEntry] = field(default_factory=list)
    max_entries: int = 10
    rolling_summary: str = ""


class ContextManager:
    """
    Manages conversation context, history, and memory injection.
    Provides sliding window history with automatic summarization.
    """

    def __init__(self):
        self.config = get_config()
        self._qwen_stack: ContextStack = ContextStack()
        self._deep_local_stack: ContextStack = ContextStack()
        self._persistent_context_cache: Dict[str, Any] = {}

    # ==================== CONTEXT STACK ====================

    def push_message(
        self,
        role: str,
        text: str,
        lane: str = "qwen",
        max_chars: int = 1800,
    ) -> None:
        """
        Push a message to the context stack.

        Args:
            role: "user" or "sage"
            text: Message text
            lane: "qwen" or "deep-local"
            max_chars: Maximum characters per message
        """
        # Truncate if needed
        if len(text) > max_chars:
            text = text[:max_chars].rstrip() + " ...[truncated]"

        entry = ContextEntry(
            role=role.lower(),
            text=text,
            timestamp=datetime.now(),
        )

        # Select appropriate stack
        if lane == "deep-local":
            stack = self._deep_local_stack
        else:
            stack = self._qwen_stack

        stack.entries.append(entry)

        # Apply sliding window with summarization
        self._apply_sliding_summary(stack)

        # Sync to state
        self._sync_stack_to_state(stack, lane)

    def get_context(
        self,
        lane: str = "qwen",
        include_summary: bool = True,
        max_chars: int = 2800,
    ) -> Optional[str]:
        """
        Get formatted context string for injection.

        Args:
            lane: "qwen" or "deep-local"
            include_summary: Include rolling summary
            max_chars: Maximum total characters

        Returns:
            Formatted context string or None if disabled
        """
        runtime_cfg = self.config.runtime

        # Check if history is enabled
        if lane == "deep-local":
            enabled = getattr(runtime_cfg, 'deep_local_history_context_enabled', True)
            stack = self._deep_local_stack
        else:
            enabled = getattr(runtime_cfg, 'qwen_history_context_enabled', True)
            stack = self._qwen_stack

        if not enabled:
            return None

        # Build context sections
        sections = []

        # Add rolling summary
        if include_summary and stack.rolling_summary:
            sections.append(stack.rolling_summary)

        # Add date and identity anchors
        sections.append(self._build_anchors())

        # Add recent messages
        recent_entries = stack.entries[-stack.max_entries:]
        if recent_entries:
            lines = []
            for entry in recent_entries:
                role_label = "User" if entry.role == "user" else "Sage"
                lines.append(f"{role_label}: {entry.text}")

            history_block = "\n".join(lines)

            # Truncate if needed
            if len(history_block) > max_chars:
                history_block = history_block[:max_chars].rstrip() + "\n...[truncated]"

            sections.append(f"[RECENT SESSION HISTORY]\n{history_block}")

        if not sections:
            return None

        return "\n\n".join(sections)

    def clear_history(self, lane: str = "qwen") -> None:
        """Clear conversation history for a lane."""
        if lane == "deep-local":
            self._deep_local_stack.entries = []
            self._deep_local_stack.rolling_summary = ""
        else:
            self._qwen_stack.entries = []
            self._qwen_stack.rolling_summary = ""

        self._sync_stack_to_state(
            self._deep_local_stack if lane == "deep-local" else self._qwen_stack,
            lane,
        )

    def _apply_sliding_summary(self, stack: ContextStack) -> None:
        """Apply sliding window with automatic summarization."""
        max_live_messages = stack.max_entries * 2  # user + sage pairs

        # Trigger summarization when buffer exceeds threshold
        while len(stack.entries) > max_live_messages and len(stack.entries) >= 16:
            # Summarize oldest 8 turns (16 messages)
            oldest_entries = stack.entries[:16]

            # Generate summary bullets
            bullets = self._summarize_entries(oldest_entries, max_bullets=8)

            # Merge with existing summary
            stack.rolling_summary = self._merge_summary(
                stack.rolling_summary,
                bullets,
                max_bullets=18,
            )

            # Remove summarized entries
            stack.entries = stack.entries[16:]

    def _summarize_entries(
        self,
        entries: List[ContextEntry],
        max_bullets: int = 8,
    ) -> List[str]:
        """Summarize a batch of entries into bullet points."""
        bullets = []
        seen = set()

        # Keywords that indicate important content
        keywords = [
            "because", "since", "therefore", "decide", "decision",
            "plan", "budget", "timeline", "due", "payment",
        ]

        for entry in entries:
            if not entry.text:
                continue

            # Extract first sentence
            sentence = re.split(r"(?<=[.!?])\s+", entry.text, maxsplit=1)[0].strip()
            if len(sentence) < 18:
                sentence = entry.text
            if len(sentence) > 180:
                sentence = sentence[:180].rstrip() + "..."

            # Score importance
            score = sum(1 for kw in keywords if kw in sentence.lower())

            # Skip low-importance entries if we have enough bullets
            if score == 0 and len(bullets) >= 3:
                continue

            role_label = "User" if entry.role == "user" else "Sage"
            bullet = f"{role_label}: {sentence}"

            # Dedupe
            dedupe_key = bullet.lower()
            if dedupe_key in seen:
                continue

            seen.add(dedupe_key)
            bullets.append(bullet)

            if len(bullets) >= max_bullets:
                break

        if not bullets:
            return ["User and Sage discussed ongoing facts and decisions for this session."]

        return bullets

    def _merge_summary(
        self,
        existing: str,
        new_bullets: List[str],
        max_bullets: int = 18,
    ) -> str:
        """Merge new bullets into existing summary."""
        merged = []
        seen = set()

        # Extract existing bullets
        for line in existing.splitlines():
            text = line.strip()
            if text.startswith("- "):
                item = text[2:].strip()
                key = item.lower()
                if item and key not in seen:
                    seen.add(key)
                    merged.append(item)

        # Add new bullets
        for item in new_bullets:
            text = re.sub(r"\s+", " ", item).strip(" -")
            key = text.lower()
            if text and key not in seen:
                seen.add(key)
                merged.append(text)

        # Keep most recent
        merged = merged[-max(4, min(max_bullets, 24)):]

        if not merged:
            return ""

        header = "[ROLLED FACTS & DECISIONS SUMMARY]"
        return header + "\n" + "\n".join("- " + row for row in merged)

    def _sync_stack_to_state(self, stack: ContextStack, lane: str) -> None:
        """Sync stack to global state."""
        if lane == "deep-local":
            state.deep_local_session_history = [
                {"role": e.role, "text": e.text, "at": e.timestamp.strftime("%Y-%m-%d %H:%M:%S")}
                for e in stack.entries
            ]
            state.deep_local_session_summary = stack.rolling_summary
        else:
            state.qwen_session_history = [
                {"role": e.role, "text": e.text, "at": e.timestamp.strftime("%Y-%m-%d %H:%M:%S")}
                for e in stack.entries
            ]
            state.qwen_session_summary = stack.rolling_summary

    # ==================== PERSISTENT CONTEXT ====================

    def get_persistent_context(self) -> Optional[str]:
        """
        Get persistent context from configured file.

        Returns:
            Context string or None if not enabled
        """
        runtime_cfg = self.config.runtime

        if not getattr(runtime_cfg, 'deep_persistent_context_enabled', False):
            return None

        # Check if perm memory is engaged
        if not state.perm_memory_ui_engaged:
            return None

        raw_path = getattr(runtime_cfg, 'deep_persistent_context_file', '')
        if not raw_path:
            return None

        # Check cache
        cache_key = str(Path(raw_path).resolve())
        cached = self._persistent_context_cache.get(cache_key)

        if cached:
            # Check if file changed
            try:
                mtime = Path(raw_path).stat().st_mtime
                if cached.get('mtime') == mtime:
                    return cached.get('context')
            except Exception:
                pass

        # Load file
        context = self._load_persistent_file(raw_path)

        if context:
            # Cache it
            try:
                mtime = Path(raw_path).stat().st_mtime
            except Exception:
                mtime = None

            self._persistent_context_cache[cache_key] = {
                'path': cache_key,
                'mtime': mtime,
                'context': context,
                'refreshed_at': datetime.now().strftime("%Y-%m-%d %H:%M:%S"),
                'loaded_chars': len(context),
            }

        return context

    def _load_persistent_file(self, raw_path: str) -> Optional[str]:
        """Load and format persistent context file."""
        try:
            path = Path(raw_path)

            if not path.is_absolute():
                path = Path.cwd() / path

            if not path.exists() or not path.is_file():
                logger.warning(f"Persistent context file not found: {path}")
                return None

            max_chars = getattr(self.config.runtime, 'deep_persistent_context_max_chars', 6000)
            max_chars = max(1200, min(max_chars, 50000))

            content = path.read_text(encoding='utf-8', errors='replace')

            # Strip any nested [ATTACHED FILE] wrapper
            match = re.search(r"\[ATTACHED FILE\]\s*\npath=[^\n]+\ncontent:\n([\s\S]+)", content, flags=re.IGNORECASE)
            if match:
                content = match.group(1).strip()

            # Truncate if needed
            if len(content) > max_chars:
                content = content[:max_chars].rstrip() + "\n...[truncated]"

            formatted = (
                "[PERSISTENT CORE MEMORY FILE]\n"
                f"path={path.name}\n"
                "Use this as long-term family context when relevant.\n"
                f"{content}"
            )

            return formatted

        except Exception as e:
            logger.error(f"Load persistent context failed: {e}", exc_info=True)
            return None

    def should_inject_persistent_context(self, user_text: str) -> bool:
        """
        Determine if persistent context should be injected for this prompt.

        Args:
            user_text: User's input text

        Returns:
            True if persistent context should be injected
        """
        low = user_text.strip().lower()

        if not low:
            return False

        # Don't inject for conversation recall prompts
        recall_markers = [
            "earlier", "previous", "previously", "we found", "you said",
            "that list", "same as before", "again", "from earlier",
        ]
        if any(marker in low for marker in recall_markers):
            return False

        # Strong family/context markers
        family_markers = [
            "our", "we", "family", "bajka", "beba", "mike", "nada", "mirjana",
            "property manager", "kinvillage", "fraser health", "doctor", "clinic",
```

```python
# -*- coding: utf-8 -*-
"""
Sage Stream Handler
Manages streaming responses with chunking and TTS integration.
"""

import logging
import re
import queue
from typing import Optional, Callable, List, Any
from dataclasses import dataclass

logger = logging.getLogger(__name__)


@dataclass
class StreamState:
    """State for a streaming session."""
    buffer: str = ""
    spoken: bool = False
    chunks_sent: int = 0
    total_chars: int = 0


class StreamHandler:
    """
    Handles streaming responses with intelligent chunking.
    Integrates with TTS for real-time speech synthesis.
    """

    def __init__(self):
        self._states: dict = {}

    def create_callbacks(
        self,
        session_id: str,
        emit_fn: Callable[[str], None],
        min_emit_chars: int = 110,
    ) -> tuple:
        """
        Create streaming callbacks for a session.

        Args:
            session_id: Unique session identifier
            emit_fn: Function to emit chunks (e.g., to UI or TTS)
            min_emit_chars: Minimum characters before emitting

        Returns:
            Tuple of (stream_cb, flush_cb, has_spoken_fn)
        """
        state = StreamState()
        self._states[session_id] = state

        def stream_cb(chunk: str) -> None:
            """Callback for each stream chunk."""
            if not chunk:
                return

            state.buffer += chunk
            state.total_chars += len(chunk)
            self._emit_if_ready(state, emit_fn, min_emit_chars)

        def flush_cb() -> None:
            """Flush remaining buffer."""
            self._emit_if_ready(state, emit_fn, min_emit_chars, force=True)

        def has_spoken_fn() -> bool:
            """Check if anything was spoken."""
            return state.spoken

        return stream_cb, flush_cb, has_spoken_fn

    def _emit_if_ready(
        self,
        state: StreamState,
        emit_fn: Callable[[str], None],
        min_chars: int,
        force: bool = False,
    ) -> None:
        """Emit buffered content if ready."""
        while True:
            buf = state.buffer

            if not buf:
                return

            # Force emit everything
            if force:
                chunk = buf.strip()
                state.buffer = ""
                if chunk:
                    try:
                        emit_fn(chunk)
                        state.spoken = True
                        state.chunks_sent += 1
                    except Exception as e:
                        logger.error(f"Emit failed: {e}")
                return

            # Try to emit on sentence boundaries
            match = re.search(r"(.+?[.!?…])(?:\s+|$)", buf, flags=re.DOTALL)

            if match:
                chunk = match.group(1).strip()
                state.buffer = buf[match.end():]

                if len(chunk) >= 14:
                    try:
                        emit_fn(chunk)
                        state.spoken = True
                        state.chunks_sent += 1
                    except Exception as e:
                        logger.error(f"Emit failed: {e}")
                continue

            # Emit on character threshold
            if len(buf) >= max(60, min_chars):
                # Split on word boundary
                split_at = buf.rfind(" ")
                if split_at <= 0:
                    split_at = min_chars

                chunk = buf[:split_at].strip()
                state.buffer = buf[split_at:].lstrip()

                if chunk:
                    try:
                        emit_fn(chunk)
                        state.spoken = True
                        state.chunks_sent += 1
                    except Exception as e:
                        logger.error(f"Emit failed: {e}")
                continue

            # Not ready yet
            return

    def cleanup(self, session_id: str) -> None:
        """Clean up a session."""
        if session_id in self._states:
            del self._states[session_id]


# Global stream handler instance
stream_handler = StreamHandler()


def get_stream_handler() -> StreamHandler:
    """Get the global stream handler instance."""
    return stream_handler
```

```python
# -*- coding: utf-8 -*-
"""
Sage Escalation Policy
Determines when to escalate from fast/local models to deep/cloud models.
"""

import logging
import re
from typing import Dict, Any, Optional, Tuple
from dataclasses import dataclass

logger = logging.getLogger(__name__)


@dataclass
class PolicyConfig:
    """Configuration for escalation policy."""
    enabled: bool = True
    low_confidence_threshold: float = 0.55
    min_length_for_trigger: int = 90
    min_length_for_strong_phrase: int = 60
    weak_executor_min_chars: int = 24


class OrchestratorEscalationPolicy:
    """
    Determines when to escalate from executor (fast/local) to deep model.
    """

    # Complexity trigger phrases
    COMPLEXITY_TRIGGERS = [
        "analyze", "analysis", "strategy", "tradeoff", "compare", "comparison",
        "plan", "roadmap", "architecture", "design", "diagnose", "root cause",
        "multi", "layered", "step by step", "pros and cons", "risk", "risks",
    ]

    # Strong phrases that always trigger deep
    STRONG_PHRASES = [
        "step by step",
        "pros and cons",
        "root cause",
        "tradeoff",
        "compare",
    ]

    def __init__(self, config: Optional[PolicyConfig] = None):
        self.config = config or PolicyConfig()

    @classmethod
    def from_config(cls, config_dict: Dict[str, Any]) -> 'OrchestratorEscalationPolicy':
        """Create policy from config dictionary."""
        cfg = PolicyConfig(
            enabled=config_dict.get('enabled', True),
            low_confidence_threshold=float(config_dict.get('low_confidence_threshold', 0.55)),
            min_length_for_trigger=int(config_dict.get('min_length_for_trigger', 90)),
            min_length_for_strong_phrase=int(config_dict.get('min_length_for_strong_phrase', 60)),
            weak_executor_min_chars=int(config_dict.get('weak_executor_min_chars', 24)),
        )
        return cls(cfg)

    def decide(
        self,
        raw_text: str,
        orchestrator_data: Optional[Dict[str, Any]],
        executor_reply: str,
    ) -> Tuple[bool, str]:
        """
        Decide whether to escalate to deep model.

        Args:
            raw_text: Original user input
            orchestrator_data: Data from orchestrator LLM (if available)
            executor_reply: Response from fast/local executor

        Returns:
            Tuple of (should_escalate, reason)
        """
        if not self.config.enabled:
            # Fallback logic when policy is disabled
            return self._fallback_decide(raw_text, orchestrator_data, executor_reply)

        lowered = raw_text.lower()
        text_len = len(raw_text.strip())

        # Check orchestrator data first
        if isinstance(orchestrator_data, dict):
            # Explicit needs_deep flag
            if orchestrator_data.get('needs_deep') is True:
                return True, "orchestrator.needs_deep"

            # Low confidence
            confidence = orchestrator_data.get('confidence')
            if confidence is not None:
                try:
                    if float(confidence) < self.config.low_confidence_threshold:
                        return True, f"orchestrator.low_confidence:{confidence}"
                except (ValueError, TypeError):
                    pass

        # Check for complexity triggers
        trigger_hits = sum(1 for trigger in self.COMPLEXITY_TRIGGERS if trigger in lowered)

        # Strong phrases
        for phrase in self.STRONG_PHRASES:
            if phrase in lowered and text_len >= self.config.min_length_for_strong_phrase:
                return True, "complexity_strong_phrase"

        # Multiple triggers
        if trigger_hits >= 2 and text_len >= self.config.min_length_for_trigger:
            return True, "complexity_trigger"

        # Weak executor response
        weak_executor = self._is_weak_executor(executor_reply)
        if weak_executor:
            return True, "weak_executor"

        return False, "executor_sufficient"

    def _is_weak_executor(self, reply: str) -> bool:
        """Check if executor response is weak/inadequate."""
        if not reply:
            return True

        reply = reply.strip()

        # Warning/error responses
        if reply.startswith("WARNING:") or reply.startswith("[WARNING]"):
            return True

        # Too short
        if len(reply) < self.config.weak_executor_min_chars:
            return True

        return False

    def _fallback_decide(
        self,
        raw_text: str,
        orchestrator_data: Optional[Dict[str, Any]],
        executor_reply: str,
    ) -> Tuple[bool, str]:
        """Fallback decision logic when policy is disabled."""
        lowered = raw_text.lower()
        text_len = len(raw_text.strip())

        # Check orchestrator data
        if isinstance(orchestrator_data, dict):
            if orchestrator_data.get('needs_deep') is True:
                return True, "orchestrator.needs_deep"

            confidence = orchestrator_data.get('confidence')
            if confidence is not None:
                try:
                    if float(confidence) < 0.55:
                        return True, f"orchestrator.low_confidence:{confidence}"
                except (ValueError, TypeError):
                    pass

        # Complexity triggers
        trigger_hits = sum(1 for trigger in self.COMPLEXITY_TRIGGERS if trigger in lowered)

        for phrase in self.STRONG_PHRASES:
            if phrase in lowered and text_len >= 60:
                return True, "complexity_strong_phrase"

        if trigger_hits >= 2 and text_len >= 90:
            return True, "complexity_trigger"

        # Weak executor
        weak = self._is_weak_executor(executor_reply)
        reason = "weak_executor" if weak else "executor_sufficient"

        return weak, reason


# Global policy instance (created on demand)
_policy_instance: Optional[OrchestratorEscalationPolicy] = None


def get_escalation_policy() -> OrchestratorEscalationPolicy:
    """Get the global escalation policy instance."""
    global _policy_instance

    if _policy_instance is None:

        _policy_instance = OrchestratorEscalationPolicy.from_config(esc_cfg if isinstance(esc_cfg, dict) else {})

    return _policy_instance
```

```python
# -*- coding: utf-8 -*-
"""
Sage Orchestrator
Main orchestration logic for routing, model selection, and response generation.
"""

import logging
import re
from datetime import datetime
from typing import Dict, List, Optional, Any, Callable, Tuple
from pathlib import Path

from ..core.state import state
from ..core.config import get_config
from ..core.utils import _strip_internal_warning_artifacts
from .models import get_model_client, ModelResponse
from .context import get_context_manager
from .streaming import get_stream_handler
from .policy import get_escalation_policy

logger = logging.getLogger(__name__)


class Orchestrator:
    """
    Main orchestrator for Sage AI responses.
    Handles routing, model selection, context injection, and response generation.
    """

    def __init__(self):
        self.config = get_config()
        self.model_client = get_model_client()
        self.context_manager = get_context_manager()
        self.stream_handler = get_stream_handler()
        self.escalation_policy = get_escalation_policy()

    def process(
        self,
        user_text: str,
        mode: str = "admin",
        stream_callback: Optional[Callable[[str], None]] = None,
        force_lane: Optional[str] = None,
    ) -> str:
        """
        Process user input and generate response.

        Args:
            user_text: User's input text
            mode: "admin" or "chat"
            stream_callback: Optional callback for streaming chunks
            force_lane: Force specific model lane (deep, deep-local, fast)

        Returns:
            Generated response text
        """
        text = user_text.strip()

        if not text:
            return ""

        logger.debug(f"Orchestrator processing: mode={mode}, lane={force_lane}")

        # Check for calculator/logic intercept
        calc_result = self._check_calculator(text, force_lane)
        if calc_result:
            state.context_stack_push(text, calc_result)
            return calc_result

        # Handle forced lanes
        if force_lane:
            return self._handle_forced_lane(text, force_lane, stream_callback)

        # Route based on mode
        if mode.lower() == "admin":
            return self._handle_admin_mode(text)
        else:  # chat mode
            return self._handle_chat_mode(text, stream_callback)

    def _check_calculator(self, text: str, lane_hint: Optional[str]) -> Optional[str]:
        """Check if input is calculator/logic intent."""
        # Extract intent text
        value = text.strip()

        # Check for calc prefix
        if re.match(r"^\s*calc\b", value.lower()):
            query = re.sub(r"^\s*calc\b\s*[:\-]?\s*", "", value, flags=re.IGNORECASE).strip()
            if query:
                return self._call_bitnet_logic(query)

        # Check for math/logic patterns
        if self._looks_like_calc_intent(value):
            query = re.sub(r"^\s*calc\b\s*[:\-]?\s*", "", value, flags=re.IGNORECASE).strip()
            if query:
                return self._call_bitnet_logic(query)

        return None

    def _looks_like_calc_intent(self, text: str) -> bool:
        """Check if text looks like calculator/logic intent."""
        low = text.lower()

        # Calc prefix
        if re.match(r"^\s*calc\b", low):
            return True

        # Reasoning markers
        reasoning_markers = (
            "calculate", "compute", "solve", "simplify", "equation",
            "formula", "volume", "area", "probability", "logic",
        )
        if any(marker in low for marker in reasoning_markers):
            return True

        # Math patterns
        if re.search(r"\d\s*[+\-*/^]\s*\d", text):
            return True

        if re.search(r"[()^]|\*\*", text) and re.search(r"\d", text):
            return True

        return False

    def _call_bitnet_logic(self, query: str) -> str:
        """Call BitNet logic engine for calculation."""
        bitnet_prompt = (
            "Solve this mathematical or logical request accurately. "
            "Return the final answer clearly and include brief steps only when helpful.\n\n"
            f"Request: {query}"
        )

        try:
            # Try to import bitnet adapter
            from bitnet_mcp_adapter import bitnet_logic
            response = str(bitnet_logic(bitnet_prompt) or "").strip()

            if response:
                return response
            return "[WARNING] BitNet returned an empty response."

        except Exception as e:
            return f"[WARNING] BitNet reasoning failed: {e}"

    def _handle_forced_lane(
        self,
        text: str,
        lane: str,
        stream_callback: Optional[Callable[[str], None]],
    ) -> str:
        """Handle explicitly forced model lane."""
        logger.debug(f"Forced lane: {lane}")

        # Build context
        context = self._build_context(text, lane)

        if lane == "deep-local":
            # Call active deep-local profile
            response = self._call_deep_local(text, context, stream_callback)
        elif lane == "deep":
            # Call deep raw
            response = self._call_deep_raw(text, context, stream_callback)
        else:
            # Call standard model
            response = self._call_model(lane, text, context)

        # Sanitize and return
        response = _strip_internal_warning_artifacts(response)
        state.context_stack_push(text, response)

        return response

    def _handle_admin_mode(self, text: str) -> str:
        """Handle admin mode processing."""
        # Admin mode should use fast/local models for commands
        # Commands are handled by router, this is for chat fallback

        context = self._build_context(text, "fast")
        response = self._call_model("fast", text, context)
        response = _strip_internal_warning_artifacts(response)

        state.context_stack_push(text, response)
        return response

    def _handle_chat_mode(
        self,
        text: str,
        stream_callback: Optional[Callable[[str], None]],
    ) -> str:
        """Handle chat mode processing with escalation."""
        # Prefer deep-local first
        context = self._build_context(text, "deep-local")

        # Try deep-local
        local_response = self._call_deep_local(text, context, stream_callback)
        local_text = _strip_internal_warning_artifacts(local_response)

        if local_text and not local_text.startswith("WARNING:"):
            state.context_stack_push(text, local_text)
            return local_text

        # Fallback to deep
        logger.debug("Deep-local failed, trying deep")
        deep_response = self._call_deep_raw(text, context, stream_callback)
        deep_text = _strip_internal_warning_artifacts(deep_response)

        if deep_text and not deep_text.startswith("WARNING:"):
            state.context_stack_push(text, deep_text)
            return deep_text

        # Final fallback to fast
        fast_response = self._call_model("fast", text, context)
        fast_text = _strip_internal_warning_artifacts(fast_response)

        state.context_stack_push(text, fast_text)
        return fast_text or "I didn't get a response from the model."

    def _build_context(
        self,
        user_text: str,
        lane: str,
    ) -> Optional[str]:
        """Build context for injection."""
        context_parts = []

        # Persistent context (if enabled and relevant)
        if self.context_manager.should_inject_persistent_context(user_text):
            persistent = self.context_manager.get_persistent_context()
            if persistent:
                context_parts.append(persistent)

        # History context
        history = self.context_manager.get_context(lane=lane)
        if history:
            context_parts.append(history)

        # File attachment context
        file_context = self._get_attached_file_context()
        if file_context:
            context_parts.append(file_context)

        if not context_parts:
            return None

        return "\n\n".join(context_parts)

    def _get_attached_file_context(self) -> Optional[str]:
        """Get any attached file context."""
        pending = state.consume_qwen_pending_file_context()

        if pending:
            return pending.get('context')

        last = state.peek_qwen_last_file_context()
        if last:
            return last.get('context')

        return None

    def _call_model(
        self,
        role: str,
        prompt: str,
        extra_context: Optional[str],
    ) -> str:
        """Call a model by role name."""
        models_cfg = self.config.models
        profile = None

        if role == "fast":
            profile = models_cfg.fast
        elif role == "deep":
            profile = models_cfg.deep
        elif role == "orchestrator":
            profile = models_cfg.orchestrator
        else:
            logger.warning(f"Unknown model role: {role}")
            profile = models_cfg.fast

        response = self.model_client.call(
            profile=profile,
            prompt=prompt,
            extra_context=extra_context,
            stream=False,
        )

        if response.error:
            logger.error(f"Model call error ({role}): {response.error}")
            return f"[WARNING] Model call failed: {response.error}"

        if response.warning:
            logger.warning(f"Model warning ({role}): {response.warning}")

        return response.text

    def _call_deep_local(
        self,
        prompt: str,
        context: Optional[str],
        stream_callback: Optional[Callable[[str], None]],
    ) -> str:
        """Call active deep-local profile."""
        active_name = self.config.active_deep_local
        models_cfg = self.config.models

        profile = models_cfg.__dict__.get(active_name)

        if not profile:
            return f"[WARNING] Deep-local profile not found: {active_name}"

        if not getattr(profile, 'enabled', False):
            return f"[WARNING] Deep-local profile disabled: {active_name}"

        # Build system prompt
        speaker_name = self.config.runtime.active_speaker_name

        system_prompt = (
            f"[Active Speaker: {speaker_name}]\n"
            "You are a local household assistant. "
            "Answer directly and concisely. "
            "Use general knowledge unless explicitly asked to answer from context."
        )

        if context:
            system_prompt += (
                " If [CONTEXT] is supplied and the question asks for context/file details, "
                "treat [CONTEXT] as authoritative for that answer."
            )

        response = self.model_client.call(
            profile=profile,
            prompt=prompt,
            system_prompt=system_prompt,
            extra_context=context,
            stream=bool(stream_callback),
            stream_callback=stream_callback,
        )

        if response.error:
            return f"[WARNING] Deep-local call failed: {response.error}"

        return response.text

    def _call_deep_raw(
        self,
        prompt: str,
        context: Optional[str],
        stream_callback: Optional[Callable[[str], None]],
    ) -> str:
        """Call deep raw chat (cloud)."""
        runtime_cfg = self.config.runtime

        if not getattr(runtime_cfg, 'deep_raw_chat_enabled', False):
            return "[WARNING] deep_raw_chat_enabled is off in config."

        profile = self.config.models.deep

        # Build system prompt
        system_prompt = (
            "You are Sage, an authentic, adaptive collaborator. "
            "Respond directly and conversationally."
        )

        response = self.model_client.call(
            profile=profile,
            prompt=prompt,
            system_prompt=system_prompt,
            extra_context=context,
            stream=True,  # Always stream for deep raw
            stream_callback=stream_callback,
        )

        if response.error:
            return f"[WARNING] Deep raw call failed: {response.error}"

        return response.text


# Global orchestrator instance
orchestrator = Orchestrator()


def get_orchestrator() -> Orchestrator:
    """Get the global orchestrator instance."""
    return orchestrator

def process_input(

    stream_callback: Optional[Callable[[str], None]] = None,

    """Process user input through the orchestrator."""
    return get_orchestrator().process(user_text, mode, stream_callback)
```

```python
# -*- coding: utf-8 -*-
"""
Sage Theme Manager
Handles UI styling, colors, and fonts.
"""

import tkinter as tk
from typing import Dict, Any
from dataclasses import dataclass


@dataclass
class ColorPalette:
    """Color palette for the UI theme."""
    # Background
    bg_primary: str = "#000000"
    bg_secondary: str = "#1a1a1a"
    bg_tertiary: str = "#0d0d0d"

    # Text
    text_primary: str = "#00bbff"
    text_secondary: str = "#00ff00"
    text_accent: str = "#ff9900"
    text_error: str = "#ff3333"
    text_success: str = "#00ff66"
    text_muted: str = "#666666"

    # Buttons
    btn_primary: str = "#0047ab"
    btn_primary_fg: str = "#ffffff"
    btn_secondary: str = "#000000"
    btn_secondary_fg: str = "#00ff00"
    btn_accent: str = "#cc7a00"
    btn_accent_fg: str = "#ffffff"

    # Status
    status_ready: str = "#00ff00"
    status_processing: str = "#ff9900"
    status_error: str = "#ff3333"
    status_lane: str = "#ff3333"

    # Scrollbar
    scrollbar_bg: str = "#003300"
    scrollbar_active: str = "#005500"
    scrollbar_trough: str = "#003300"


@dataclass
class FontConfig:
    """Font configuration for the UI."""
    family: str = "Courier New"
    header_size: int = 13
    body_size: int = 12
    status_size: int = 11
    small_size: int = 10
    weight_normal: str = "normal"
    weight_bold: str = "bold"


class ThemeManager:
    """
    Manages UI theme, colors, and fonts.
    Provides centralized styling for all UI components.
    """

    def __init__(self):
        self.colors = ColorPalette()
        self.fonts = FontConfig()
        self._root: tk.Tk = None

    def set_root(self, root: tk.Tk):
        """Set the root window for theme application."""
        self._root = root
        self._apply_global_settings()

    def _apply_global_settings(self):
        """Apply global Tkinter settings."""
        if not self._root:
            return

        # Handle Windows DPI scaling
        try:
            import ctypes
            try:
                ctypes.windll.shcore.SetProcessDpiAwareness(1)
            except Exception:
                ctypes.windll.user32.SetProcessDPIAware()
        except Exception:
            pass

        # Set default background
        self._root.configure(bg=self.colors.bg_primary)

    def get_header_font(self) -> tuple:
        """Get header font tuple."""
        return (self.fonts.family, self.fonts.header_size, self.fonts.weight_bold)

    def get_body_font(self) -> tuple:
        """Get body text font tuple."""
        return (self.fonts.family, self.fonts.body_size, self.fonts.weight_normal)

    def get_status_font(self) -> tuple:
        """Get status bar font tuple."""
        return (self.fonts.family, self.fonts.status_size, self.fonts.weight_normal)

    def get_small_font(self) -> tuple:
        """Get small font tuple."""
        return (self.fonts.family, self.fonts.small_size, self.fonts.weight_normal)

    def get_small_bold_font(self) -> tuple:
        """Get small bold font tuple."""
        return (self.fonts.family, self.fonts.small_size, self.fonts.weight_bold)

    def style_output_widget(self, widget) -> None:
        """Style a text widget for output display."""
        widget.configure(
            bg=self.colors.bg_primary,
            fg=self.colors.text_primary,
            font=self.get_body_font(),
            insertbackground=self.colors.text_primary,
            selectbackground=self.colors.bg_secondary,
            selectforeground=self.colors.text_primary,
        )

    def style_entry_widget(self, widget) -> None:
        """Style an entry widget for command input."""
        widget.configure(
            bg=self.colors.bg_secondary,
            fg=self.colors.btn_primary,
            font=self.get_body_font(),
            insertbackground=self.colors.text_primary,
            relief=tk.FLAT,
        )

    def style_button_primary(self, widget) -> None:
        """Style a primary action button."""
        widget.configure(
            bg=self.colors.btn_primary,
            fg=self.colors.btn_primary_fg,
            font=self.get_small_font(),
            relief=tk.FLAT,
            activebackground="#0055cc",
            activeforeground=self.colors.btn_primary_fg,
        )

    def style_button_secondary(self, widget) -> None:
        """Style a secondary action button."""
        widget.configure(
            bg=self.colors.bg_primary,
            fg=self.colors.text_secondary,
            font=self.get_small_font(),
            relief=tk.SOLID,
            bd=1,
            activebackground="#001a0d",
            activeforeground="#33ff88",
            highlightthickness=0,
        )

    def style_button_accent(self, widget) -> None:
        """Style an accent button."""
        widget.configure(
            bg=self.colors.bg_primary,
            fg=self.colors.text_accent,
            font=self.get_small_font(),
            relief=tk.SOLID,
            bd=1,
            activebackground="#2b1400",
            activeforeground="#ffb84d",
            highlightthickness=0,
        )

    def style_status_label(self, widget, status_type: str = "ready") -> None:
        """Style a status label."""
        widget.configure(
            bg=self.colors.bg_primary,
            font=self.get_status_font(),
            anchor="w",
        )

        color_map = {
            "ready": self.colors.status_ready,
            "processing": self.colors.status_processing,
            "error": self.colors.status_error,
            "lane": self.colors.status_lane,
        }

        fg_color = color_map.get(status_type, self.colors.status_ready)
        widget.configure(fg=fg_color)

    def style_scrollbar(self, widget) -> None:
        """Style a scrollbar widget."""
        widget.configure(
            bg=self.colors.scrollbar_bg,
            activebackground=self.colors.scrollbar_active,
            troughcolor=self.colors.scrollbar_trough,
        )

    def get_window_geometry(self, screen_w: int, screen_h: int) -> str:
        """Calculate optimal window geometry."""
        target_w = min(max(600, int(screen_w * 0.38)), min(700, max(600, screen_w - 80)))
        target_h = min(max(500, int(screen_h * 0.52)), min(560, max(500, screen_h - 120)))
        pos_x = max(0, (screen_w - target_w) // 2)
        pos_y = max(0, (screen_h - target_h) // 2)

        return f"{target_w}x{target_h}+{pos_x}+{pos_y}"

    def get_min_size(self) -> tuple:
        """Get minimum window size."""
        return (560, 460)


# Global theme manager instance
theme_manager = ThemeManager()


def get_theme() -> ThemeManager:
    """Get the global theme manager instance."""
    return theme_manager
```

```python
class CommandEntry(tk.Entry):
    """
    Custom entry widget for command input.
    Includes context menu and keyboard shortcuts.
    """

    def __init__(
        self,
        parent,
        on_submit: Optional[Callable[[str], None]] = None,
        **kwargs
    ):
        super().__init__(parent, **kwargs)

        self.on_submit = on_submit
        self.theme = get_theme()

        # Apply theme
        self.theme.style_entry_widget(self)

        # Bind events
        self.bind("<Return>", self._on_enter)
        self.bind("<Control-v>", self._on_paste)
        self.bind("<Control-a>", self._on_select_all)
        self.bind("<Button-2>", self._block_middle_click)
        self.bind("<Shift-Insert>", self._block_middle_click)
        self.bind("<<PasteSelection>>", self._block_middle_click)
        self.bind("<Button-3>", self._show_context_menu)

        # Context menu
        self.context_menu = tk.Menu(parent, tearoff=0)
        self.context_menu.add_command(label="Cut", command=self._cut)
        self.context_menu.add_command(label="Copy", command=self._copy)
        self.context_menu.add_command(label="Paste", command=self._paste)

    def _on_enter(self, event):
        """Handle Enter key press."""
        if self.on_submit:
            text = self.get().strip()
            if text:
                self.on_submit(text)
        return "break"

    def _on_paste(self, event):
        """Handle Ctrl+V paste."""
        self.event_generate("<<Paste>>")
        return "break"

    def _on_select_all(self, event):
        """Handle Ctrl+A select all."""
        self.select_range(0, tk.END)
        self.icursor(tk.END)
        return "break"

    def _block_middle_click(self, event):
        """Block middle-click paste."""
        return "break"

    def _show_context_menu(self, event):
        """Show context menu on right-click."""
        try:
            self.context_menu.tk_popup(event.x_root, event.y_root)
        finally:
            self.context_menu.grab_release()

    def _cut(self):
        """Cut selected text."""
        self.event_generate('<<Cut>>')

    def _copy(self):
        """Copy selected text."""
        self.event_generate('<<Copy>>')

    def _paste(self):
        """Paste text."""
        self.event_generate('<<Paste>>')

    def clear(self):
        """Clear the entry."""
        self.delete(0, tk.END)

    def get_text(self) -> str:
        """Get entry text."""
        return self.get().strip()


class OutputDisplay(scrolledtext.ScrolledText):
    """
    Custom scrolled text widget for output display.
    Includes context menu and read-only mode.
    """

    def __init__(
        self,
        parent,
        **kwargs
    ):
        super().__init__(parent, state='disabled', wrap=tk.WORD, **kwargs)

        self.theme = get_theme()

        # Apply theme
        self.theme.style_output_widget(self)
        self.theme.style_scrollbar(self.vbar)

        # Context menu
        self.context_menu = tk.Menu(parent, tearoff=0)
        self.context_menu.add_command(label="Copy", command=self._copy)

        # Bind events
        self.bind("<Button-3>", self._show_context_menu)
        self.bind("<Control-c>", self._on_copy)
        self.bind("<Control-a>", self._on_select_all)

    def _show_context_menu(self, event):
        """Show context menu on right-click."""
        try:
            self.context_menu.tk_popup(event.x_root, event.y_root)
        finally:
            self.context_menu.grab_release()

    def _on_copy(self, event):
        """Handle Ctrl+C copy."""
        self._copy()
        return "break"

    def _on_select_all(self, event):
        """Handle Ctrl+A select all."""
        self.tag_add("sel", "1.0", "end")
        return "break"

    def _copy(self):
        """Copy selected text or all text."""
        try:
            text = self.selection_get()
        except tk.TclError:
            # No selection - copy all visible text
            text = self.get("1.0", tk.END).strip()

        if text:
            self.master.clipboard_clear()
            self.master.clipboard_append(text)

    def append(self, text: str, scroll: bool = True) -> None:
        """Append text to output."""
        self.configure(state='normal')
        self.insert(tk.END, text + "\n\n")

        if scroll:
            self.see(tk.END)

        self.configure(state='disabled')

    def clear(self) -> None:
        """Clear all output."""
        self.configure(state='normal')
        self.delete('1.0', 'end')
        self.configure(state='disabled')

    def set_readonly(self, readonly: bool = True) -> None:
        """Set read-only mode."""
        self.configure(state='disabled' if readonly else 'normal')
```
